설치 준비

파일 및 폴더를 추출하고 작업 폴더에 복사

  1. 폴더 만들기[Magento_Root]/app/code/Brightcove/Upload
  2. 아카이브 파일 (코드 파일)을 다운로드합니다.
  3. 파일 / 폴더 압축 해제
  4. 압축을 푼 파일의 내용을 1 단계의 디렉토리로 드롭 / 이동

확장 및 플러그인 활성화

다음을 활성화합니다.

  • php bin/magento module:enable Brigtcove_Upload
  • php bin/magento setup:upgrade
  • php bin/magento cache:clean
  • php bin/magento setup:static-content:deploy
  • php bin/magento cache:flush

Brightcove 모듈 자격 증명 구성

이 부분을 완료하려면 확장이 사용하는 API에 대한 클라이언트 자격 증명을 가져와야합니다. 자격 증명을 만드는 방법에 대한 자세한 내용은 API 인증 자격 증명관리를 참조하십시오 . 자격 증명에 필요한 최소 권한은 다음과 같습니다.

  1. 스토어 -> 구성 -> 브라이트코브 (왼쪽 탭) -> 브라이트코브 비디오 구성으로 이동합니다.
  2. Brightcove 구성에 대한 세부 정보를 입력하십시오.
    • 클라이언트 ID : REST 클라이언트 / 사용자 ID
    • 클라이언트 시크릿 : REST 클라이언트 비밀 / 암호.
    • API 기본 URL :
    • ID : Brightcove 스토어 ID
    • 카테고리 비디오 슬롯 : 비디오 슬롯 수를 범주 열로 설정하려면
    • 마스터 비디오 슬롯 : 비디오 슬롯 수를 마스터 열로 설정하려면
    • 변형 비디오 슬롯: 비디오 슬롯 수를 Variant Column으로 설정하려면
    • 이메일 주소: 일괄 업로드 상태와 관련된 이메일을 보내려면
  3. 모든 세부 정보가 올바르게 제공되면 구성 페이지를저장합니다 . 이제 사용자는 Brightcove 사용자 정의 모듈에 액세스 할 수 있습니다.

확장 기능 비활성화

Brightcove 확장을 비활성화해야하는 경우 다음 명령을 사용하십시오.

  1. php bin/magento module:disable Brightcove_Upload
  2. php bin/magento setup:upgrade
  3. php bin/magento cache:clean
  4. php bin/magento setup:static-content:deploy
  5. php bin/magento cache:flush
