비디오 업로드

  1. 브라이트코브 모듈 메뉴에서비디오추가페이지를 엽니다.
  2. 비디오를 추가 할 Store View를 선택합니다.
  3. 비디오 제목을 추가합니다 (필수).
  4. 로컬 비디오 파일을 찾습니다 (필수).
  5. 로컬 썸네일 이미지 파일을 찾습니다 (필수).
  6. 로컬 포스터 이미지 파일을 찾습니다 (선택 사항).
  7. 동영상에 대해 쉼표로 구분 된 태그를 입력합니다 (선택 사항).
  8. 비디오에 대한 설명을 입력합니다 (선택 사항).
  9. 저장을 클릭합니다 .

참고: 스토어 보기는 비디오 메타데이터에 태그로 저장됩니다. 이 태그는 할당 페이지에서 현지화를 구현하는 데 사용됩니다.

카테고리 및 제품에 비디오 할당

비디오는 카테고리 및 제품에 할당 될 수 있습니다. 이 섹션에서는 비디오를 카테고리에 할당하는 방법과 다음 섹션에서 비디오를 제품에 할당하는 방법을 살펴 봅니다.

카테고리에 할당

  1. 브라이트코브 모듈 메뉴에서범주에 비디오할당페이지를 엽니다.
  2. 초기 페이지에는 현재 할당된 비디오를 보여주는 범주 목록이 표시되며 다음 링크가 있습니다. 할당 해제또는변화기존 비디오 과제뿐만 아니라양수인카테고리에 동영상.

    참고 사항:

    • 지역화 필터를 구현할 수있는 범위 선택기
    • 동영상 ID 위로 마우스를 가져 가면 동영상의 미리보기 이미지가 표시됩니다.
  3. 할당할 카테고리옆의 할당을 클릭하여 할당할 비디오 그리드를 엽니다.
  4. 찾고있는 비디오가 보이지 않으면 비디오 제목으로 검색하여 찾을 수 있습니다.
  5. 현지화 드롭 다운 메뉴를 사용하여 원하는 언어를 얻으십시오.
  6. 비디오열에서 다음을 수행합니다.
    • 선택한 비디오의 플레이어를 선택하십시오
    • 선택기를 사용하여 비디오 슬롯 선택
    슬롯 값의 수는 구성 값에서 생성됩니다.
    .
  7. 비디오가 이미 이 카테고리에 할당되어 있으면 주황색 테두리와할당 취소옵션이 표시됩니다.

제품에 할당

  1. 브라이트코브 모듈 메뉴에서제품에 비디오할당페이지를 엽니다.
  2. 카테고리와 마찬가지로 다음을 수행할 수 있습니다. 양수인비디오 또는할당 해제또는변화이미 할당된 비디오. 동영상 ID 위로 마우스를 가져 가서 미리보기 이미지를 표시 할 수도 있습니다.
  3. 할당링크를 클릭하여 비디오를 할당합니다. 그러면 비디오 그리드가 나타납니다.
  4. 찾고있는 비디오가 보이지 않으면 비디오 제목으로 검색하여 찾을 수 있습니다.
  5. 현지화 드롭 다운 메뉴를 사용하여 원하는 언어를 얻으십시오.
  6. 비디오열에서 다음을 수행합니다.
    • 선택한 비디오의 플레이어를 선택하십시오
    • 선택기를 사용하여 비디오 슬롯 선택
    슬롯 값의 수는 구성 값에서 생성됩니다.
    .
  7. 동영상이 이미 이 제품에 할당된 경우 주황색 테두리와할당 취소옵션이 표시됩니다.

임베드 코드 생성

  1. 브라이트코브 모듈 메뉴에서임베드 코드 생성 페이지로 이동합니다.
  2. 사용할 플레이어와 비디오를 선택한 다음 코드가져오기를 클릭하여 플레이어의 기본 (iframe) 또는 고급 (페이지 내) 코드를 보고 복사합니다.
  3. 인 페이지 스크립트를 필요로하는 방식으로 플레이어를 맞춤 설정하지 않는 한 기본 소스 코드를 사용하는 것이 좋습니다.

Brightcove 구성

Brightcove 구성을 업데이트하거나 추가 할 수 있습니다.

  1. 저장 -> 브라이트코브 비디오 구성으로 이동합니다 .
  2. 클라이언트 ID, 클라이언트 시크릿, API Base URL (액세스 토큰 가져 오기 용) 및 Store ID는 설치 중에 이미 채워져 있어야하지만 필요한 경우 이러한 값을 업데이트 할 수 있습니다.
  3. 카테고리, 마스터 및 변형에 대한 비디오 슬롯 수를 설정할 수도 있습니다.
  4. 마지막으로 대량 데이터 가져 오기 옵션 (다음 섹션에서 설명)을 사용하려면 쉼표로 구분 된 이메일 주소 목록을 입력하여 대량 가져 오기와 관련된 알림을받습니다.

대량 가져 오기 데이터

Brightcove에 업로드 할 비디오 목록, 비디오 범주 할당 및 비디오 제품 할당을 포함하여 대량으로 가져올 수있는 여러 종류의 데이터가 있습니다.

  1. 시스템 -> 가져오기로 이동하여 대량 가져오기 페이지를엽니다.
  2. 수행하려는 가져 오기 유형을 선택하십시오.
  3. 다음은 Brightcove에 비디오 대량 업로드 페이지의 샘플입니다.
  4. 가져올 샘플 CSV 파일이없는 경우 샘플 다운로드 링크를 사용하여 다운로드 할 수 있습니다.
  5. 가져오기 동작을 선택합니다 .
    • 추가 / 업데이트 -기존 기록 추가 및 교체
    • 바꾸다 -기존 레코드 교체
    • 지우다 -기존 기록 삭제
  6. 찾아보기버튼을 사용하여 CSV 파일을 선택합니다.
  7. 데이터확인을 클릭하여 데이터가 유효한지 확인합니다. (CSV 파일에 유효한 레코드가없는 경우 가져 오기 옵션이 표시되지 않습니다.)
  8. 가져오기버튼을 클릭하여 데이터를 가져옵니다. 성공 메시지가 표시되어야합니다.
