페이지 내용

    세일즈포스 상거래 클라우드 사용자 가이드

    이 문서는 브라이트코브 Salesforce Commerce Cloud 커스텀 카트리지를 사용하여 비디오 파일을 브라이트코브 클라우드/서버에 업로드하고 필요에 따라 이러한 파일을 스토어프론트 (제품 세부 정보 페이지, 카테고리 랜딩 페이지, 제품 목록 페이지 등) 에 게시하기 위한 단계별 지침을 제공합니다.

    다중 언어 지원

    확장은 비즈니스 관리자 인터페이스에 대한 여러 UI 언어 지원을 제공합니다. 참고: 이 버전의 확장 프로그램은 UI 언어용 영어와 프랑스어를 지원합니다.

    비디오 추가

    다음은 비디오를 추가하는 단계입니다.

    1. 판매자 도구 - > 브라이트코브 - 비디오 > 추가
    2. 메뉴에서언어를선택합니다. 기본값은 사이트에 대해 선택된 기본 로케일입니다.
    3. 비디오 필드 옆에 있는선택옵션을 클릭합니다.
    4. 나타나는 대화 상자에서 비디오 파일을 선택하십시오.
    5. 썸네일 필드 옆에 있는선택옵션을 클릭하면 썸네일 파일을 선택할 수 있는 유사한 팝업 대화상자가 나타납니다.
    6. 포스터 필드 옆에 있는선택옵션을 클릭하면 포스터 파일을 선택할 수 있는 유사한 팝업 대화 상자가 나타납니다.
    7. 양식을 제출하여 수집 프로세스를 시작합니다.
    8. 제출 후 양식에 성공 메시지의 유효성을 검사합니다:
    9. 브라이트코브 스튜디오에 로그인하고 Media 모듈로 이동하여 비디오가 성공적으로 수집되었는지 확인할 수 있습니다.
    10. 로케일 코드를 확인하기 위해 비디오를 클릭하여 세부 정보를 확인할 수도 있습니다.

    비디오 할당 - 범주

    먼저 카테고리에 비디오를 할당하는 단계를 살펴 보겠습니다.

    카테고리에 비디오 할당

    1. 판매자 도구 - > 브라이트코브 - 비디오 > 할당 (카테고리)으로 이동합니다.
    2. 비디오는 로케일에 할당됩니다.
      • 로케일 관련 데이터는 드롭 다운에서 로케일을 변경하여 볼 수 있습니다:

        현재 지원되는 유일한 로케일은영어 (미국)와프랑스어 (프랑스)뿐입니다.

      • 비디오 썸네일은 비디오 ID에 커서를 올려 놓으면 볼 수 있습니다.
    3. 비디오 할당옵션을 클릭하여 하나 이상의 비디오를 할당합니다.
    4. 이미 할당된 비디오가 할당된 비디오 그리드에서 선택한 대로 표시됩니다.
    5. 할당된 슬롯은 같은 로케일의 다른 비디오에 사용할 수 없습니다.
    6. 플레이어와 슬롯은 각 비디오 타일에서 사용할 수 있는 드롭다운을 사용하여 변경할 수 있습니다.
    7. 슬롯이 다른 다른 비디오를 선택합니다.
    8. 선택한 비디오를 할당하려면 [비디오 저장] 버튼을 클릭합니다.
    9. 할당된 비디오 및 슬롯을 확인합니다.
    10. 카테고리 객체의 데이터를 확인합니다.
    11. 판매자 도구 - > 카탈로그로이동합니다.
    12. 카탈로그를 선택합니다.
    13. 범주/하위 범주를 선택합니다.
    14. 편집을선택합니다.
    15. 범주 속성을선택합니다.
    16. 브라이트코브 섹션까지 아래로 스크롤합니다.
    17. 하위 범주에 비디오를 할당하려면 범주를 클릭하십시오.
    18. 하위 범주 목록이 나타납니다.
    19. 3-16단계를 수행하여 하위 범주에 비디오를 할당합니다.

    비디오 임베드

    비디오 임베딩 단계를 통해 ISML 페이지에서 비디오를 렌더링하는 코드를 얻을 수 있습니다.

    1. 판매자 도구 - > 브라이트코브 - 비디오 > 삽입으로이동합니다.
    2. 원하는 비디오 및 플레이어를 선택하고코드 가져오기를클릭합니다.
    3. 코드와 함께 팝업 대화 상자가 나타납니다. 코드 블록을 선택하면 클립 보드에 복사됩니다.
    4. 코드를 콘텐츠 자산, 콘텐츠 슬롯 또는 ISML 템플릿에 붙여넣어 비디오를 렌더링합니다.

    대량 업로드

    대량 업로드는.csv 파일을 업로드하여 대량 수량으로 비디오를 수집 할 수 있습니다. 이 작업은.csv 파일을 처리하고 비디오를 브라이트코브 계정으로 인제스트합니다. 샘플 벌크업로드 파일은 메타 디렉토리 아래의 zip 파일에 첨부됩니다.

    참고videofile , videoposter과videothumbnail공개적으로 액세스 할 수있는 경로에서 사용할 수 있어야합니다.
    1. 판매자 도구 - > 브라이트코브 - > 대량 업로드로이동합니다.
    2. .csv 파일을 선택합니다.
    3. 제출을 클릭합니다.
    4. 관리 - > 작업으로이동합니다.
    5. 대량 업로드작업을 선택하고실행을누릅니다.
    6. 작업 완료 상태를 확인합니다.
    7. 브라이트코브 계정에서 비디오 객체를 확인합니다.
    8. 처리 된 파일대량 업로드작업이 이동됩니다 bulkUpload_processed아래 폴더온도예배 규칙서.

    대량 할당

    대량 할당은.csv 파일을 업로드하여 대량 수량의 카테고리, 하위 카테고리 및 제품에 비디오를 할당 할 수 있습니다. 대량 할당은.csv 파일을 업로드하여 대량 수량의 카테고리, 하위 카테고리 및 제품에 비디오를 할당 할 수 있습니다. 범주 및 제품에 대한 샘플 BulkAssign 파일은메타디렉토리 아래의 zip 파일에 첨부되어 있습니다.

    상품 파일 데이터는 다음과 같습니다.

    이 작업은 동일한 실행에서 카테고리 및 제품에 대한 csv 파일을 처리합니다. 이 작업은 디렉토리에서 제품 및 범주를 확인하고 파일을 처리합니다.

    범주

    3. .csv 파일을 선택합니다.

    5. 파일이대량 할당 범주폴더 아래의Temp디렉토리에 업로드됩니다.
    6. 관리 - > 작업으로이동합니다.
    7. 대량 할당작업을 선택하고실행을누릅니다.
    8. 작업 완료 상태를 확인합니다.
    9. 할당된 비디오를 확인합니다.
    10. 대량 할당 작업에 의해 처리된 파일은 Temp 디렉터리 아래의대량 할당 카테고리 처리폴더로 이동됩니다.

    제품

    2. 제품을선택합니다.
    3. .csv 파일을 선택합니다.
    4. 제출을 클릭합니다.
    5. 파일이Temp디렉토리에 있는BulkAssignProduct폴더에 업로드됩니다.
    6. 관리 - > 작업으로이동합니다.
    7. 대량 할당작업을 선택하고실행을누릅니다.
