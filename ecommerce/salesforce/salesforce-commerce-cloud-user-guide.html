다국어 지원

이 확장은 비즈니스 관리자 인터페이스에 대한 여러 UI 언어 지원을 제공합니다. 참고: 이 버전의 확장은 UI 언어에 대해 영어와 프랑스어를 지원합니다.

비디오 추가

다음은 동영상을 추가하는 단계입니다.

  1. 판매자 도구 -> 브라이트코브 -> 비디오 추가로 이동합니다.
  2. 선택하다언어메뉴에서(기본값은 사이트에 대해 선택된 기본 로케일입니다):
  3. 비디오 필드 옆에 있는선택옵션을 클릭합니다.
  4. 표시되는 대화 상자에서 비디오 파일을 선택합니다.
  5. 축소판 필드 옆에 있는선택옵션을 클릭하면 축소판 파일을 선택할 수 있는 유사한 팝업 대화 상자가 나타납니다.
  6. 포스터 필드 옆에 있는선택옵션을 클릭하면 포스터 파일을 선택할 수 있는 유사한 팝업 대화 상자가 나타납니다.
  7. 수집 프로세스를 시작하려면 양식을 제출하세요.
  8. 제출 후 양식에서 성공 메시지를 확인합니다.
  9. Brightcove Studio에 로그인하고 미디어 모듈로 이동하여 비디오에서 성공적으로 수집되었는지 확인할 수 있습니다.
  10. 로케일 코드를 확인하기 위해 비디오를 클릭하여 세부 정보를 표시 할 수도 있습니다.

비디오 할당-카테고리

먼저 카테고리에 동영상을 할당하는 단계를 살펴 보겠습니다.

카테고리에 비디오 할당

  1. 판매자 도구 -> 브라이트코브 -> 비디오 할당 (카테고리) 로 이동합니다 .
  2. 비디오는 로케일에 할당됩니다.
    • 로케일 관련 데이터는 드롭 다운에서 로케일을 변경하여 볼 수 있습니다.

      현재 지원되는 유일한 로케일은영어 (미국)와 프랑스 ( 프랑스) 뿐입니다.

    • 동영상 ID에 커서를 올려 놓으면 동영상 미리보기 이미지를 볼 수 있습니다.
  3. 비디오할당옵션을 클릭하여 하나 이상의 비디오를 할당합니다.
  4. 이미 할당 된 비디오는 할당 비디오 그리드에서 선택된 것으로 나타납니다.
  5. 할당 된 슬롯은 동일한 로케일의 다른 동영상에 사용할 수 없습니다.
  6. 플레이어와 슬롯은 각 비디오 타일에서 사용 가능한 각 드롭 다운을 사용하여 변경할 수 있습니다.
  7. 슬롯이 다른 다른 비디오를 선택합니다.
  8. 비디오 저장 버튼을 클릭하여 선택한 비디오를 할당합니다.
  9. 할당 된 비디오 및 슬롯을 확인합니다.
  10. 범주 개체의 데이터를 확인합니다.
  11. 판매자 도구 -> 카탈로그로 이동합니다 .
  12. 카탈로그를 선택하십시오.
  13. 카테고리 / 하위 카테고리를 선택합니다.
  14. 편집을 선택합니다 .
  15. 범주 속성을 선택합니다 .
  16. Brightcove 섹션까지 아래로 스크롤합니다.
  17. 하위 범주에 비디오를 할당하려면 범주를 클릭하십시오.
  18. 하위 범주 목록이 나타납니다.
  19. 3-16 단계에 따라 비디오를 하위 범주에 할당합니다.

비디오 삽입

동영상 삽입 단계를 사용하면 ISML 페이지에서 동영상을 렌더링하는 코드를 얻을 수 있습니다.

  1. 판매자 도구 -> 브라이트코브 -> 비디오 임베드로 이동합니다 .
  2. 원하는 비디오 및 플레이어를 선택하고 코드가져오기를 클릭합니다 .
  3. 코드와 함께 팝업 대화 상자가 나타납니다. 코드 블록을 선택하면 클립 보드에 복사됩니다.
  4. 콘텐츠 자산, 콘텐츠 슬롯 또는 ISML 템플릿에 코드를 붙여 넣어 비디오를 렌더링합니다.

대량 업로드

대량 업로드를 사용하면.csv 파일을 업로드하여 대량으로 비디오를 수집할 수 있습니다. 작업은.csv 파일을 처리하고 비디오를 브라이트코브 계정으로 인제스트합니다. 샘플 bulkUpload 파일은 meta 디렉토리 아래의 zip 파일에 첨부됩니다.

videofile , videoposter및videothumbnail공개적으로 액세스할 수 있는 경로에서 사용할 수 있어야 합니다.
  1. 판매자 도구 -> 브라이트코브 -> 대량 업로드로 이동합니다 .
  2. .csv 파일을 선택합니다.
  3. 제출을 클릭하십시오.
  4. 관리 -> 작업으로 이동합니다 .
  5. 대량 업로드작업을 선택하고실행을 클릭합니다 .
  6. 작업 완료 상태를 확인하십시오.
  7. Brightcove 계정에서 비디오 개체를 확인하십시오.
  8. 대량 업로드작업에 의해 처리된 파일은 Temp디렉터리의 Bulkupload_processed폴더로 이동됩니다.

범주

  1. 판매자 도구 -> 브라이트코브 -> 대량 할당으로 이동합니다 .
  2. 범주를 선택합니다 .

  4. 제출을 클릭합니다 .
  5. 파일은 대량 AssignCategory폴더 아래의 Temp디렉토리에 업로드됩니다.
  6. 관리 -> 작업으로 이동합니다 .
  7. 대량 할당작업을 선택하고 [ 실행] 을 클릭합니다 .
  8. 작업 완료 상태를 확인하십시오.
  9. 할당 된 비디오를 확인하십시오.
  10. 대량 할당 작업에서 처리된 파일은 임시디렉터리 아래의 대량 AssignCategory_PROCESSED폴더로 이동합니다.
