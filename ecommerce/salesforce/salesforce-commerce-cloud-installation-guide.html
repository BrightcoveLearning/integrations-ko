사용자 지정 사이트 기본 설정 구성

  1. 선택대지드롭다운 메뉴에서:
  2. 판매자 도구 -> 사이트 기본 설정 -> 사용자 지정 기본 설정으로 이동합니다 .
  3. 브라이트코브를 선택합니다 .
  4. 다음 값을 추가하십시오.
    • Brightcove 계정 ID
    • Brightcove 클라이언트 ID
    • Brightcove 클라이언트 비밀
    • 사이트 카탈로그
    보다 API 인증 자격 증명 관리클라이언트 ID 및 비밀을 생성하는 방법에 대한 정보는 자격 증명에는 다음 권한이 필요합니다.
  5. Brightcove 슬롯 제한 (1-11)을 추가합니다. 기본값은 11입니다.
    • 마스터 제품용 슬롯 ( 마스터및 m1-m10 )
    • 변형 제품용 슬롯 ( v1 - v11 )
    • 카테고리 슬롯 ( c1 - c11 )

    슬롯 제한 5는 구성을 위해 다음 슬롯을 제공합니다.

    • 마스터 제품 ( 마스터및 m1-m4 )
    • 변형 제품 ( v1 - v5 )
    • 카테고리 ( c1 - c5 )

ISML 파일에 Brightcove 비디오 렌더링 포함

SFRA 카트리지 –app_brightcove

  1. 비디오 렌더링은 아래 ISML 파일에 대해 지원됩니다.
    • productDetails.isml
    • catLanding.isml
    • searchResultsNoDecorator.isml
  2. 포함 사항modules.isml : 
    <isinclude template="modules">
  3. 에서 비디오를modules.isml렌더링할 태그를 포함합니다.
    • 마스터 제품 비디오<isM1> - m1슬롯에 할당된 비디오를 렌더링합니다.
    • 변형 제품 비디오<isV1> - v1슬롯에 할당된 비디오를 렌더링합니다.
    • 카테고리 비디오<isC1> - C1슬롯에 할당된 비디오를 렌더링합니다.

SiteGenesis 카트리지 –int_brightcove

  1. 비디오 렌더링은 아래 ISML 파일에 대해 지원됩니다.
    • productcontents.isml
    • producttopcontent.isml
    • categoryproducthits.isml
    • catlanding.isml
  2. 포함 사항modules.isml : 
    <isinclude template="modules">
  3. 에서 비디오를modules.isml렌더링할 태그를 포함합니다.
    • 마스터 제품 비디오<isM1> - m1슬롯에 할당된 비디오를 렌더링합니다.
    • 변형 제품 비디오<isV1> - v1슬롯에 할당된 비디오를 렌더링합니다.
    • 카테고리 비디오<isC1> - C1슬롯에 할당된 비디오를 렌더링합니다.
