개요

브라이트코브는콜렉티브 eCDN을 사용하여 브라이트코브 플레이어에게 VOD (비디오 온 디맨드 비디오) 및 브라이트코브 라이브 이벤트를 제공하기 위한 통합을 만들었습니다 .

이 통합은 다음 두 부분으로 구성됩니다.

  • Brightcove에서 Kollective로 비디오를 동기화하는 데 사용되는 백엔드 애플리케이션

  • Kollective eCDN을 사용하여 비디오 자산을 재생할 수있는 Brightcove Player 용 Kollective 플러그인

이 항목은 콘텐츠를 동기화하는 데 사용되는 Kollective의 서버 측 앱에 중점을 둡니다.

프로세스 흐름

VOD 또는 Live 콘텐츠를 재생하려면 다음 프로세스 흐름을 사용하십시오.

  1. Syndicator 앱에서 VOD 또는 Live 자산을 선택합니다.

  2. Syndicator 앱을 사용하여 콘텐츠 자산을 Kollective에 동기화합니다.

  3. 콘텐츠를 재생하려면 웹 페이지에 Kollective 플러그인과 함께 Brightcove Player를 포함하십시오.

시작하기

콘텐츠를 동기화하려면 다음 단계를 따르세요.

  1. 아직 완료하지 않은 경우 Kollective eCDN 배포 가이드의 설정 지침을 따르십시오 .

  2. 사용자 역할 검토

  3. VOD와 라이브 콘텐츠를 동기화하려면 다음을 참조하세요.

    • 비디오 동기화
    • 라이브 이벤트 동기화
  4. 신디케이터 로그 확인
  5. 플레이어 코드 삽입

사용자 역할 검토

Kollective 백엔드 신디케이터 애플리케이션은 두 가지 다른 사용자 역할을 지원합니다.

  • 관리자 역할-관리자는 다른 사용자 (관리자 및 사용자 역할)를 만들고, 회사 설정에 액세스하고, 신디케이터 도구를 실행하고, 동기화 로그를 확인할 수 있습니다. 신디케이터 앱의 모든 섹션에 액세스 할 수 있습니다.

  • 사용자 역할-이러한 사용자는 신디케이터 도구를 실행하고 동기화 로그를 확인할 수 있지만 회사 설정을 편집 할 수는 없습니다. 비디오 신디케이터, 라이브 이벤트 신디케이터및신디케이터로그에 액세스할 수 있지만 사용자관리또는설정페이지.

관리자

관리자는 다른 사용자 (관리자 및 사용자 역할)를 만들고, 회사 설정에 액세스하고, 신디케이터 도구를 실행하고, 동기화 로그를 확인할 수 있습니다.

계정 설정

관리자 사용자는설정섹션에 액세스하여 계정 정보를 설정할 수 있습니다. 계정 설정을 관리하려면 다음 단계를 따르세요.

  1. 관리자로 Kollective 신디케이터 앱에 로그인합니다.

  2. 왼쪽 탐색에서설정을 선택합니다 .

  3. 설정섹션에서 회사 계정에 대한 정보를 입력할 수 있습니다.

    필드 세부 정보는 다음과 같습니다.

    • 임차인 이름 -회사 명

    • 기술 -설명

    • 담당자 이름 -연락처 이름

    • 이메일 연락처 -귀하의 연락처 이메일 주소

    • Brightcove 계정 ID -Brightcove의 Video Cloud Studio에서 얻을 수 있습니다.

    • Brightcove 클라이언트 키 -Brightcove의 Video Cloud Studio에서 얻을 수 있습니다.

      Brightcove 클라이언트 비밀 -Brightcove의 Video Cloud Studio에서 얻을 수 있습니다.

      Kollective가 Brightcove의 콘텐츠에 액세스하려면 클라이언트 자격 증명을 제공해야합니다. Video Cloud Studio의 관리 섹션에서 생성 할 수 있습니다. 다음 권한을 선택해야합니다.

      • CMS: 재생 목록 읽기, 재생 목록 읽기 / 쓰기, 비디오 읽기, 비디오 읽기 / 쓰기
      • 동적 인제스트: 파일 생성, 푸시
      • 플레이어: 읽기, 읽기 / 쓰기
      • 사이: 읽기, 읽기 / 쓰기

      Brightcove 클라이언트 키 및 클라이언트 암호를 얻는 방법에 대한 자세한 내용은 다음을 참조하십시오.

      • API 인증 자격 증명 관리
      • 또는 다음 도구를 사용할 수 있습니다. 클라이언트 자격 증명 만들기

    • Brightcove Live API 키 -이를 얻으려면 Brightcove 지원에 문의하십시오.

    • Kollective 테넌트 ID -이 정보는 계정 관리자로부터받습니다.

    • 콜렉티브 서비스 토큰 -이 정보는 계정 관리자로부터받습니다.

    • 자동 동기화 허용 -특정 시간에 모든 콘텐츠를 동기화하려면이 항목을 선택하십시오. 그렇지 않으면 선택하지 않은 상태로 두십시오. 자세한 내용은 다음 단계를 참조하십시오.

  4. 자동 동기화허용을 선택하면 다음과 같은 세부 정보가 표시됩니다.

    이렇게하면 구성 가능한 시간 간격으로 동기화 작업이 자동으로 실행됩니다.

    • 작업을 실행할 시간 간격을 입력합니다.(예: 1시간마다 작업을 실행합니다).
    • 작업이 처음으로 실행되는 시간을 선택하십시오.
    • 들어가다동기화할 최대 동영상작업이 실행될 때마다.
    • 지금 작업실행단추를 클릭하여 작업을 즉시 실행할 수 있습니다.

  5. 업데이트버튼을 선택하여 설정을 저장합니다.

사용자 관리

관리자 계정으로 사용자를 생성, 편집 및 삭제할 수 있습니다.

  1. 관리자로 Kollective 신디케이터 앱에 로그인합니다.

  2. 왼쪽 탐색에서 사용자관리를 선택합니다 .

  3. 회사 계정의 모든 사용자 목록이 표시됩니다.

사용자 추가

관리자 또는 사용자 역할이있는 사용자를 추가 할 수 있습니다.

  1. 사용자관리페이지에서 사용자추가단추를 선택합니다.

  2. 정보를 입력하여 관리자 또는 사용자 역할이있는 사용자를 만듭니다.

사용자 편집

기존 사용자에 대한 정보를 편집 할 수 있습니다.

  1. 사용자관리페이지의 목록에서 사용자를 선택합니다.

  2. 전자 메일을 제외한 모든 사용자 필드를 업데이트할 수있습니다.

사용자 삭제

사용자 목록에서 사용자를 삭제할 수 있습니다.

  1. 사용자관리페이지의 목록에서 사용자를 선택합니다 (로그인한 사용자 제외).

  2. 삭제버튼을 선택합니다.

  3. 로부터사용자 삭제대화 상자에서삭제단추.

비디오 동기화

현재 계정의 비디오를 Brightcove에서 Kollective로 동기화합니다.

비디오 목록

  1. 왼쪽 탐색에서콘텐츠 신디케이터를선택한 다음비디오 신디케이터를 선택합니다 .

    가장 최근에 추가 / 업데이트 된 비디오가 맨 위에있는 Video Cloud 계정의 모든 비디오 목록이 표시되어야합니다.

  2. 비디오 목록에서모두, 동기화됨또는동기화되지 않은비디오를 필터링할 수 있습니다.

  3. 목록에서 하나 이상의 비디오를 선택하고동기화버튼을 선택하여 브라이트코브에서 Kollective 로 동기화합니다.

비디오 세부 정보

  1. 목록에서 비디오를 선택하여 비디오 세부 정보 페이지를 표시합니다.

  2. 비디오 세부 정보 페이지에서 다음을 수행 할 수 있습니다.

    • 비디오 동기화
    • 비디오 상태 업데이트
    • 비디오 삭제

라이브 이벤트 동기화

현재 계정에서 Brightcove에서 Kollective로 라이브 이벤트를 동기화합니다.

비디오 목록

  1. 왼쪽 탐색에서콘텐츠 신디케이터를선택한 다음라이브 이벤트 신디케이터를 선택합니다 .

    가장 최근에 추가 / 업데이트 된 비디오가 맨 위에있는 Video Cloud 계정의 모든 라이브 이벤트 목록이 표시되어야합니다.

  2. 라이브 이벤트 목록에서모두, 라이브, 대기또는완료된라이브 이벤트를 필터링할 수 있습니다.

  3. 목록에서 하나 이상의 라이브 이벤트를 선택하고동기화버튼을 선택하여 브라이트코브에서 Kollective 로 동기화합니다.

라이브 이벤트 세부 정보

  1. 목록에서 라이브 이벤트를 선택하여 세부 정보 페이지를 표시합니다.

  2. 라이브 이벤트 세부 정보 페이지에서 다음을 수행 할 수 있습니다.

    • 라이브 이벤트 동기화
    • 라이브 이벤트 상태 업데이트
    • 라이브 이벤트 삭제

신디케이터 로그 확인

신디케이터 로그는 계정의 비디오 및 라이브 이벤트 동기화에 대한 성공, 오류 및 경고 세부 정보를 제공합니다.

  1. 왼쪽 탐색에서신디케이터 로그를 선택합니다 .

    최신 계정이 맨 위에있는 현재 계정에 대한 동기화 로그 목록이 표시됩니다.

  2. 동기화 로그 목록에서모두, 오류, 정보또는경고를 필터링할 수있습니다.

플레이어 코드 삽입

웹 페이지에 플레이어 및 관련 콘텐츠 자산을 포함하는 방법은 변경되지 않습니다.

  1. Brightcove의 Video Cloud Studio에서 미디어 모듈로 이동합니다.

  2. 사용할 비디오를 선택하고 [ 게시] 및 [포함]단추를 선택합니다.

  3. Kollective 플러그인을 사용하도록 구성된 플레이어를 선택하십시오.

  4. 표준또는고급코드를 복사하여 웹 페이지에 추가합니다.

    기본적으로 플레이어 토큰은 콘텐츠를 동기화한 후 6시간 후에 만료됩니다. 만료되면 콘텐츠를 재생할 수 없습니다. 만료 시간을 변경하려면 브라이트코브 지원팀에 문의하십시오.
