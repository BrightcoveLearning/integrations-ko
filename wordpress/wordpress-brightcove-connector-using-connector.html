페이지 내용

    워드 프레스 브라이트코브 커넥터: 커넥터 사용

    이 항목에서는 Wordpress-Brightcove 커넥터를 사용하여 Wordpress에서 비디오를 관리하고 게시하는 방법에 대해 설명합니다.

    서문

    WordPress-BrightCove 커넥터를 사용하면 WordPress에서 브라이트코브 비디오 클라우드 비디오 및 플레이어를 관리하고 WordPress 페이지에 비디오를 쉽게 삽입할 수 있습니다.

    비디오 플레이어 출판

    비디오 플레이어 게시를 통해 사용자는 브라이트코브 계정의 비디오를 WordPress 사이트에 쉽게 삽입할 수 있습니다.

    동영상을 게시하려면 새 게시물 페이지로 이동하기만 하면 됩니다. 핵심 기능에는 제목 아래와 편집기 위에 “미디어 추가”버튼이 있습니다. 브라이트코브 비디오 커넥트에서는 “브라이트코브 미디어” 버튼도 사용할 수 있습니다. 이 버튼을 클릭하면 브라이트코브 미디어 모달이 열립니다.

    여기에서 사용자는 이미지를 클릭하여 사용할 이미지를 선택할 수 있습니다. 비디오의 썸네일이 옵션과 함께 측면에 나타납니다. 특정 동영상을 찾으려면 제목, 태그 또는 설명별로 검색할 수 있습니다.

    그런 다음 사용자가 사용할 플레이어를 선택하고 자동 재생, 임베드 스타일 및 크기 조정과 같은 옵션을 사용자 지정할 수 있습니다.

    비디오를 삽입하는 데는 두 가지 옵션이 있습니다. 간단히 “게시물에 삽입”버튼을 클릭하면 게시물에 비디오를 삽입하지만 사용자는 단축 코드를 복사하여 게시물에 붙여 넣을 수 있습니다. “게시물에 삽입”버튼을 사용하는 경우 단축 코드는 일반 텍스트보기에서 계속 표시됩니다.

    동영상이 새 게시물 페이지에 나타나지만 게시되는 모습을 보려면 게시물 미리보기를 확인하십시오. 그것은 매우 다를 수 있습니다.

    재생 목록 게시

    재생목록은 단일 동영상과 똑같은 방식으로 게시할 수 있습니다. 재생 목록을 게시하려면 재생 목록을 지원하는 브라이트코브 플레이어가 있어야 합니다.

    비디오 콘텐츠 관리

    사용자는 브라이트코브에 비디오를 업로드하고 WordPress 대시보드에서 바로 메타데이터를 업데이트할 수 있습니다.

    동영상 업로드

    브라이트코브에 비디오를 업로드하려면 WordPress의 브라이트코브 관리자 페이지에서 “새로 추가”를 클릭합니다.“

    여기에서 파일을 선택하거나 업로드할 페이지로 파일을 끌어다 놓을 수 있습니다. 한 번에 여러 개의 파일을 업로드하도록 선택할 수 있습니다.

    파일이 페이지에 나타납니다. 메타데이터를 추가하려면 파일 이름을 클릭하면 옵션이 열립니다.

    준비가되면 업로드를 시작하십시오.

    “업로드” 링크를 클릭하면 새 게시물 페이지의 미디어 모달과 같은 방법으로 동영상을 업로드할 수 있습니다.

    실패

    동영상이 업로드되지 않으면 파일 권한에 문제가 있을 수 있습니다. 웹 서버에 wp-content/업로드 폴더에 대한 읽기/쓰기 권한이 있는지 확인하십시오. 동영상이 일시적으로 여기에 업로드됩니다.

    브라이트코브는 방화벽 뒤에서 수집하지 않습니다. 문제가 되면 새 비디오가 생성되지만 코드 변환되지 않습니다.

    해결 방법은 브라이트코브 계정으로 이동하여 소스 파일을 교체합니다. WordPress에서 비디오를 인제스트하려면 브라이트코브가 방화벽을 열어야 합니다.

    기존 축소판 및 포스터를 유지하는 확인란을 선택 취소하는 것을 잊지 마십시오. 그렇지 않으면 회색 상자로 유지됩니다.

    업로드가 끝나면 동영상은 WordPress에서 사용할 수 있어야합니다.

    자세한 내용은CMS 및 DAM 통합과 관련된 디버깅 네트워크 및 API 연결 문제를확인하십시오.

    메타데이터

    WordPress에서 사용자는 비디오의 이름, 간단한 설명, 긴 설명 및 태그를 업데이트 할 수 있습니다.

    새로운 미리보기 이미지, 포스터, 자막 트랙도 업로드할 수 있지만 동영상 업로드와 마찬가지로 방화벽이나 권한에 의해 차단될 수 있습니다.

    버전 1.7.0부터는비디오에 대해 미리 정의된 Video Cloud 폴더를 선택할 수도 있습니다. 이 작업은 Wordpress와 비디오 클라우드 모두에서 해당 폴더로 비디오를 이동합니다. 참고: 비디오 클라우드에 폴더를 만들어야 합니다.

    사용자 정의 필드를 포함한 기타 메타데이터 필드는 브라이트코브 스튜디오에서 편집해야 합니다.

    삭제

    사용자는 편집 페이지에서 동영상을 삭제할 수 있습니다. 확인을 위해 경고가 팝업됩니다.

    재생 목록 관리

    재생 목록은 WordPress에서 만들 수 없지만 일단 존재하면 편집 할 수 있습니다. 동영상을 추가하거나 제거하거나 목록에서 위 또는 아래로 이동할 수 있습니다.

    플레이어

    플레이어는 브라이트코브 스튜디오에서만 만들거나 편집할 수 있지만, 특정 동영상의 크기 등의 일부 측면을 게시할 때 변경할 수 있습니다. 재생 목록을 게시하려면 재생 목록 플레이어가 이미 존재해야 합니다.

    페이지 내 경험

    브라이트코브 플러그인은 페이지 내 경험을 지원합니다. 이 옵션은 모든 사용자에게 표시되지만 특정 계정에 대해 옵션을 사용할 수 있어야 합니다. 그렇지 않으면 작동하지 않습니다. 페이지 내 경험에 액세스 할 수있는 계정만 Wordpress에서 사용할 수 있습니다.

    Wordpress를 통해 페이지 내 경험을 통합하는 방법에는 비디오 경험과 재생 목록 경험이라는 두 가지가 있습니다. 동영상 경험을 통해 먼저 포함하려는 환경을 선택합니다. 사이드 바에는 사용 가능한 환경이 나열되어 있는 드롭다운이 있습니다.

    비디오를 선택하지 않고 “게시물에 삽입”을 누르면 이전에 만든 페이지 내 환경이 삽입됩니다. 또는 일부 동영상을 선택한 다음 포함하려는 페이지 내 경험을 선택하여 해당 동영상을 경험으로 전환할 수도 있습니다. 마찬가지로 재생 목록 환경을 통해 재생 목록과 경험을 선택한 다음 게시물에 삽입합니다. 재생 목록은 경험으로 변환됩니다.
