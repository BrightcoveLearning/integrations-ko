WordPress-Brightcove 커넥터: 커넥터 사용

이 항목에서는 Wordpress에서 비디오를 관리하고 게시하기위한 Wordpress-Brightcove 커넥터 사용에 대해 설명합니다.

소개

WordPress-Brightcove 커넥터를 사용하면 WordPress 내에서 Brightcove Video Cloud 비디오 및 플레이어를 관리하고 WordPress 페이지에 비디오를 쉽게 포함 할 수 있습니다.

비디오 플레이어 게시

비디오 플레이어 게시를 통해 사용자는 Brightcove 계정의 비디오를 WordPress 사이트에 쉽게 삽입 할 수 있습니다.

동영상을 게시하려면 새 게시물 페이지로 이동하기 만하면됩니다. 핵심 기능에는 제목 아래 및 편집기 위에 "미디어 추가"버튼이 있습니다. Brightcove Video Connect에는 "Brightcove Media"버튼도 있습니다. 이를 클릭하여 Brightcove 미디어 모달을 엽니 다.

여기에서 사용자는 클릭하여 사용하려는 이미지를 선택할 수 있습니다. 비디오의 썸네일이 옵션과 함께 측면에 나타납니다. 특정 비디오를 찾으려면 제목, 태그 또는 설명으로 검색 할 수 있습니다.

그런 다음 사용자는 사용하려는 플레이어를 선택하고 자동 재생, 삽입 스타일 및 크기 조정과 같은 옵션을 사용자 지정할 수 있습니다.

비디오 삽입에는 두 가지 옵션이 있습니다. "Insert into Post"버튼을 클릭하기 만하면 동영상이 게시물에 삽입되지만 사용자는 단축 코드를 복사하여 게시물에 붙여 넣을 수도 있습니다. "게시물에 삽입"버튼을 사용하면 일반 텍스트보기에 짧은 코드가 계속 표시됩니다.

동영상은 새 게시물 페이지에 표시되지만 게시 된 모습을 보려면 게시물 미리보기를 확인하세요. 매우 다를 수 있습니다.

재생 목록 게시

재생 목록은 단일 동영상과 똑같은 방식으로 게시 할 수 있습니다. 재생 목록을 게시하려면 재생 목록이 활성화 된 Brightcove 플레이어가 있어야합니다.

비디오 콘텐츠 관리

사용자는 Brightcove에 비디오를 업로드하고 WordPress 대시 보드에서 직접 메타 데이터를 업데이트 할 수 있습니다.

비디오 업로드

Brightcove에 비디오를 업로드하려면 WordPress의 Brightcove 관리 페이지에서 "새로 추가"를 클릭하십시오.

여기에서 파일을 선택하거나 업로드 할 페이지로 파일을 드래그 할 수 있습니다. 한 번에 여러 개의 파일을 선택하여 업로드 할 수 있습니다.

파일이 페이지에 나타납니다. 메타 데이터를 추가하려면 파일 이름을 클릭하면 옵션이 열립니다.

준비가되면 업로드 시작을 누르십시오.

"업로드"링크를 클릭하여 새 게시물 페이지의 미디어 모달에서 동일한 방식으로 비디오를 업로드 할 수 있습니다.

실패

동영상 업로드에 실패하면 파일 권한 문제 일 수 있습니다. 웹 서버에 wp-content / uploads 폴더에 대한 읽기 / 쓰기 권한이 있는지 확인하십시오. 동영상이 여기에 일시적으로 업로드됩니다.

Brightcove는 또한 방화벽 뒤에서 수집하지 않습니다. 이것이 문제인 경우 새 비디오가 생성되지만 트랜스 코딩되지 않습니다.

해결 방법으로 Brightcove 계정으로 이동하여 소스 파일을 교체하십시오. Brightcove가 WordPress에서 비디오를 수집하려면 방화벽을 열어야합니다.

기존 썸네일 및 포스터를 유지하려면 확인란을 선택 취소하는 것을 잊지 마십시오. 그렇지 않으면 회색 상자로 유지됩니다.

업로드가 완료되면 WordPress에서 동영상을 사용할 수 있습니다.

자세한 내용은 CMS 및 DAM 통합 관련 네트워크 및 API 연결 문제디버깅을 참조하세요 .

메타 데이터

WordPress에서 사용자는 동영상 이름, 간단한 설명, 자세한 설명 및 태그를 업데이트 할 수 있습니다.

새로운 썸네일, 포스터 및 캡션 트랙도 업로드 할 수 있지만 동영상 업로드와 마찬가지로 방화벽이나 권한에 의해 차단 될 수 있습니다.

버전 1.7.0부터비디오에 대해 미리 정의된 Video Cloud 폴더를 선택할 수도 있습니다. 이 작업은 비디오를 Wordpress 및 Video Cloud의 해당 폴더로 이동합니다. 참고: Video Cloud에서 폴더를 만들어야합니다.

사용자 정의 필드를 포함한 기타 메타 데이터 필드는 Brightcove Studio에서 편집해야합니다.

삭제

사용자는 편집 페이지에서 동영상을 삭제할 수 있습니다. 확인을 위해 경고가 나타납니다.

재생 목록 관리

재생 목록은 WordPress에서 만들 수 없지만 일단 존재하면 편집 할 수 있습니다. 목록에서 비디오를 추가 또는 제거하거나 위아래로 이동할 수 있습니다.

플레이어

플레이어는 Brightcove Studio에서만 생성 또는 편집 할 수 있지만 크기와 같은 일부 측면은 게시 될 때 특정 비디오에 대해 변경할 수 있습니다. 재생 목록을 게시하려면 재생 목록 플레이어가 이미 존재해야합니다.

인 페이지 경험

Brightcove 플러그인은 인 페이지 경험을 지원합니다. 옵션은 모든 사용자에게 표시되지만 특정 계정에 대해 옵션을 사용할 수 있어야합니다. 그렇지 않으면 작동하지 않습니다. 페이지 내 경험에 액세스 할 수있는 계정 만 Wordpress에서 사용할 수 있습니다.

워드 프레스를 통해 페이지 내 경험을 통합하는 방법에는 비디오 경험과 재생 목록 경험의 두 가지가 있습니다. 비디오 경험을 통해 먼저 포함 할 경험을 선택합니다. 사이드 바에는 사용 가능한 경험이 나열된 드롭 다운이 있습니다.

동영상을 선택하지 않고 "게시물에 삽입"을 누르면 이전에 생성 된 인 페이지 경험이 삽입됩니다. 또는 일부 비디오를 선택한 다음 포함 할 인 페이지 경험을 선택하여 해당 비디오를 경험으로 전환하십시오. 마찬가지로 재생 목록 경험을 통해 재생 목록과 경험을 선택한 다음 게시물에 삽입하십시오. 재생 목록은 경험으로 변환됩니다.
