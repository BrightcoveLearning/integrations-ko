WordPress-Brightcove 커넥터: 구성

이 항목에서는 Wordpress 용 Brightcove 커넥터 구성에 대해 설명합니다.

소개

WordPress Brightcove Video Connector를 사용하면 WordPress 내에서 Brightcove Video Cloud 비디오 및 플레이어를 관리하고 WordPress 페이지에 비디오를 쉽게 포함 할 수 있습니다.

Brightcove 계정에 연결

  1. Brightcove 설정으로 이동하고 Brightcove 계정 추가를 클릭하십시오.
  2. 소스 이름은 무엇이든 될 수 있지만 Brightcove의 계정 이름과 동일한 이름을 사용하면 쉽게 식별 할 수 있습니다.
  3. 계정 ID를 찾으려면 Brightcove Studio에 로그인하고 대시 보드의 왼쪽 상단을보십시오.
  4. 클라이언트 ID와 클라이언트 암호를 얻으려면 응용 프로그램을 등록해야합니다. API 인증으로 이동하여 새 애플리케이션 추가를 클릭하십시오.
  5. 이름과 간단한 설명을 입력하십시오.
  6. 계정을 선택하십시오. 아래 그림은 필요한 모든 권한입니다.
  7. 저장을 누르십시오.
  8. Client ID와 Client Secret이 나타납니다. 클라이언트 시크릿을 안전한 위치에 복사하여 저장하십시오. 다시 액세스 할 수 없습니다.
  9. "자격 증명 확인"을 클릭합니다. 성공하면 동영상이 표시됩니다.

수집 프로필 설정

커넥터 버전 1.4.1 및 이전에는 Video Cloud 수집 프로필을 "videocloud-default-v1"로 하드 코딩했습니다.  Brightcove 계정에 Dynamic Delivery 또는 CAE 만 활성화 된 경우이 수집 프로필이 존재하지 않고 비디오 업로드가 실패합니다. 다음은 커넥터에서 이 동작을 해결할 수 있는 몇 가지 옵션입니다.

 

  1. 커넥터에 사용 된 기본 프로필 이름을 사용하여 Video Cloud 계정에 새 수집 프로필을 만듭니다.  관리자->인제스트 프로필로 이동하여 새 프로필을 만들거나 기존 프로필을 복제하고 이름을 “videocloud-default-v1"로 지정합니다.  

 

  1. 또는 먼저 사용하려는 Video Cloud 계정의 Admin-> Ingest에서 기존 수집 프로필을 확인하여 커넥터 소스 코드의 수집 프로필 이름을 변경합니다. Show API Name을 클릭하고 기록해 둡니다.
  1. 워드프레스 서버에서 프로필을wp-content/plugins/brightcove-video-connect/includes/api/class-bc-cms-api.php편집하고 'videocloud-default-v1의' 기본 프로필 이름에서 원하는 프로필 이름으로 변경합니다. 예를 들어, 변경
    public function video_upload( $video_id, $video_url, $profile = 'videocloud-default-v1' ) {
    ...에
    public function video_upload( $video_id, $video_url, $profile = 'multi-platform-standard-static' ) {
  1. 또는 Video Cloud 계정의 기본 수집 프로필을 사용하도록 커넥터를 수정합니다. 이렇게 하려면 다음 줄을wp-content/plugins/brightcove-video-connect/includes/api/class-bc-cms-api.php편집하고 주석 처리하십시오.
    $data = array( 'profile' => sanitize_text_field( $profile ) );
    와 함께/* */ .

이제 WordPress-Brightcove 커넥터의 구성을 완료했습니다. 커넥터사용으로 진행합니다.
