SharePoint-Brightcove 커넥터 4.0/4.1: 커넥터 사용

비디오 추가 및 편집

콘텐츠 작성자는 SharePoint 커넥터를 사용하여 로컬 컴퓨터에서 비디오를 업로드하고 비디오에 대한 메타 데이터를 편집 할 수 있습니다.

비디오 추가

커넥터의 비디오 추가 섹션은 비디오 클라우드에 새 비디오를 업로드하는 데 사용됩니다. 사용자가 비디오 추가에 액세스하면 (할 수있는 권한이있는 경우) Video Cloud로 전송 될 메타 데이터를 입력 할 수있는 다음 필드가 표시됩니다.

  • 계정 선택 -이 드롭 다운 목록은 사용자가 비디오를 추가 할 권한이있는 모든 계정으로 채워집니다. 이는 계정 그룹 할당을 통해 활성화됩니다. 사용자는 업로드를 활성화하려면 계정을 선택해야합니다.
  • 비디오 파일 -비디오 파일을 상자로 드래그하거나 "찾아보기"버튼을 클릭하여 로컬 파일 시스템 탐색기를 열고 업로드 할 비디오 파일을 선택합니다. 이것은 화면을 전환하지 않고 여러 번 반복 할 수 있습니다. 이 화면에서 설정하려면 폴더 및 Brightcove 태그 다음에이 단계를 수행해야합니다. 나중에 편집 할 수도 있습니다.
  • 폴더 -비디오는 정리 목적으로 특정 폴더에 저장할 수 있습니다.
  • Brightcove 태그 -SharePoint에 저장되는 것 외에도 Video Cloud 태그 정보에 저장 될 데이터 태그입니다. 이는 일부 클라이언트가이 인터페이스에 추가하고 SharePoint에만 저장되는 (비디오 클라우드로 전송되지 않음) SharePoint 메타 데이터 태그와는 별개이며 구별됩니다.

비디오 처리가 완료되면 "비디오 편집"링크가 나타나 사용자가 메타 데이터를 추가 할 수 있습니다.

비디오 편집

비디오 편집 화면은 업로드 후 비디오 편집 링크를 통해 또는 비디오 관리로 이동하고 원하는 비디오를 클릭하여 액세스 할 수 있습니다.

사용자는 "사용 가능한 비디오"필드에 검색어를 입력하여 비디오를 검색하고 데이터 정렬 방법을 선택할 수도 있습니다. 비디오 이름 또는 "비디오 편집"링크를 클릭하면 비디오 편집 페이지로 이동합니다.

동영상이 업로드되는 Brightcove Video Cloud 계정은 변경할 수 없지만 동영상 이름 및 설명과 계정에 포함 된 사용자 정의 필드를 비롯한 대부분의 다른 정보는 변경할 수 있습니다. 이름은 파일 이름에서 자동 생성되지만 변경할 수 있습니다.

포스터와 썸네일도 생성되지만 여기서도 변경할 수 있습니다. 이미지를 변경하는 방법에는 두 가지가 있습니다. 비디오의 오른쪽 상단에있는 캡처 버튼을 사용하여 비디오에서 새 화면 캡을 캡처하거나 새 이미지를 업로드 할 수 있습니다.

텍스트 트랙 추가

텍스트 트랙은 첫 번째 업로드 또는 후속 업데이트시 모든 비디오에 추가 할 수 있습니다. 비디오에는 하나 이상의 텍스트 트랙이 연결될 수 있습니다. 사용자는 비디오에 추가 할 텍스트 트랙 유형 (파일 업로드, URL 업로드 또는 원격 URL)을 선택할 수 있습니다. "없음"을 선택하면 섹션에 필드가 표시되지 않습니다.

중대한: 단일 비디오는 파일 업로드, URL 업로드 또는 원격 URL의 혼합을 지원하지 않습니다. 각 비디오에는 한 가지 유형의 텍스트 트랙 만 연결할 수 있습니다.

파일 업로드

텍스트 트랙 파일에 대해 "파일 업로드"를 선택하면 다음 필드가 표시됩니다.

  1. 텍스트 트랙 유형 선택 -파일 업로드, URL 업로드 또는 원격 URL을 선택할 수 있습니다.
  2. 파일 선택기 -텍스트 트랙 파일 선택.
  3. 텍스트 트랙 속성 -텍스트 트랙과 관련된 속성을 설정합니다. "추가"를 클릭하면 다음 업로드 / 업데이트시 비디오 데이터에 텍스트 트랙이 추가됩니다.
    1. 언어 - 텍스트 트랙의 언어에 대한 두 글자 코드 (유효한 BCP 47 언어 태그) 입니다 (예: 영어의 경우 “en”).
    2. 상표 -자막에 사용할 수있는 여러 언어를 나열하는 메뉴와 같이 사용자에게 표시되는 트랙의 레이블입니다.
    3. 종류 -나열된 다섯 가지 지원되는 트랙 유형 중 하나
  4. 추가 / 대기열 텍스트 트랙 목록 -다음 저장시 업로드 대기중인 기존 텍스트 트랙 및 새 텍스트 트랙 목록을 표시합니다. "제거"링크를 클릭하고 비디오를 저장하여 텍스트 트랙을 삭제할 수 있습니다.

URL 업로드, 원격 URL 업로드

이러한 옵션 중 하나를 선택하면 "파일 업로드"업로드가 텍스트 트랙 파일에 대한 정규화 된 URL 경로를 넣을 텍스트 필드로 대체됩니다.

  • URL 업로드 - 텍스트 트랙 파일이 있는 공용 URL을 입력한 다음 업로드를 클릭합니다. 파일이 Video Cloud에 업로드됩니다.
  • 원격 URL -텍스트 트랙 파일이있는 공개 URL을 입력합니다. URL은 250 자 미만이어야합니다. 캡션은 원격 URL에서로드됩니다.
  1. 텍스트 트랙 유형 선택 -파일 업로드, URL 업로드 또는 원격 URL을 선택할 수 있습니다.
  2. URL 추가 -텍스트 트랙 파일의 정규화 된 URL 경로를 입력하는 필드입니다.
  3. 텍스트 트랙 속성 -텍스트 트랙과 관련된 속성을 설정할 수 있습니다. "추가"를 클릭하면 다음 업로드 / 업데이트시 비디오 데이터에 텍스트 트랙이 추가됩니다.
    1. 언어 - 텍스트 트랙의 언어에 대한 두 글자 코드 (유효한 BCP 47 언어 태그) 입니다 (예: 영어의 경우 “en”).
    2. 상표 -자막에 사용할 수있는 다양한 언어를 나열하는 메뉴와 같이 사용자에게 표시 될 트랙의 레이블입니다.
    3. 종류 -나열된 다섯 가지 지원되는 트랙 유형 중 하나.
  4. 추가 / 대기열 텍스트 트랙 목록 -다음 저장시 업로드 대기중인 기존 텍스트 트랙 및 새 텍스트 트랙 목록을 표시합니다. "제거"링크를 클릭하고 비디오를 저장하여 텍스트 트랙을 삭제할 수 있습니다.

재생 목록 및 경험 추가, 가져 오기 및 삭제

커넥터의 재생 목록 관리 섹션은 새 재생 목록을 만들고 편집 및 업데이트를 포함하여 기존 재생 목록을 찾아보고 관리하는 데 사용됩니다. 비디오와 달리 재생 목록 정보는 SharePoint에 저장되지 않고이 섹션에서 계정을 검색 할 때마다 Video Cloud에서 검색됩니다. 이러한 이유로 API를 통한 재생 목록 데이터의 가용성에 내재 된 약간의 지연이 있으며 다음 섹션에서 설명합니다.

재생 목록 랜딩 페이지 관리

재생 목록 관리 랜딩 페이지에는 재생 목록을 찾아 보거나 사용 가능한 계정에서 새 항목을 만드는 데 사용할 수있는 필드 목록이 표시됩니다.

  • 계정 선택 -이 드롭 다운 목록은 사용자가 액세스 할 수있는 모든 계정으로 채워집니다. 이는 계정 그룹 할당을 통해 활성화됩니다. 사용자는 재생 목록 목록을 보려면 계정을 선택해야합니다.
  • 사용 가능한 재생 목록 -재생 목록을 검색하려면이 필드에 용어를 입력하십시오.
  • 재생 목록 표시 열 -이 열에 재생 목록 데이터가 표시됩니다. 결과는 열 제목을 클릭하여 열별로 정렬 할 수 있습니다.
  • 재생 목록 목록 - 각 재생 목록 목록에는 다음이 포함됩니다.
    • 재생 목록 이름-이름을 클릭하면 데이터 편집을위한 재생 목록이 열립니다.
    • 재생 목록에서 활성화 된 동영상 ID 목록입니다.
    • 간단한 설명.
    • 재생 목록 유형- 수동 또는 스마트 재생 목록.
    • 재생 목록의 ID (Video Cloud 재생 목록 ID)입니다.
    • "재생 목록 편집"링크. 이 링크 (또는 이름 클릭)는 편집 할 재생 목록을 열거 나 세부 정보를 보는 데 사용됩니다.
  • 새 재생 목록 추가 -이 버튼을 클릭하면 선택한 계정에 새 재생 목록이 생성됩니다.

새로운 수동 재생 목록 추가

계정에 새 재생 목록을 추가하려면 사용자는 "새 재생 목록 추가"버튼을 클릭합니다. 그러면 재생 목록 추가 양식이 열립니다.

이동할 비디오를 선택하고 "비디오 이동"옵션을 활성화하여 위 / 아래 버튼을 사용하여 재생 목록에서 비디오를 위아래로 이동할 수 있습니다. 한 번에 여러 비디오를 이동할 수 있습니다.

  • 이름 -새 재생 목록의 이름입니다.
  • 기술 -재생 목록에 대한 간략한 설명입니다.
  • 유형 -재생 목록의 유형입니다. 이 경우 수동으로 설정해야합니다.
  • 사용 가능한 비디오 -검색 창 및 계정에서 사용 가능한 동영상 목록이 표시됩니다. 이를 통해 사용자는 재생 목록에 추가 할 비디오 선택 범위를 좁힐 수 있습니다.
  • 액션 화살표 -선택한 비디오를 재생 목록 안팎으로 이동합니다.
  • 이 재생 목록의 비디오 -재생 목록의 현재 동영상이 여기에 표시됩니다.
  • 창조하다 -변경 사항을 적용하고 새 재생 목록을 만드는 과정을 시작합니다.

새로운 스마트 재생 목록 추가

  • 이름 -새 재생 목록의 이름입니다.
  • 기술 -재생 목록에 대한 간략한 설명입니다.
  • 유형 -재생 목록의 유형입니다. 이 경우 Smart로 설정해야합니다.
  • 태그 -스마트 재생 목록의 경우 사용자는 재생 목록을 필터링하는 데 사용할 수있는 하나 이상의 태그를 선택할 수 있습니다. 첫 번째 드롭 다운에서는 사용자가 "하나 이상 포함"또는 "모두 포함"을 선택할 수 있고 두 번째 드롭 다운에서는 사용자가 스마트 재생 목록의 태그를 입력 할 수 있습니다. 선택 항목에 따라 태그가있는 모든 동영상이 표시되거나 모든 태그가 포함 된 동영상 만 표시됩니다.
  • 주문 -일치하는 동영상의 순서를 설정하는 방법을 사용자가 지정할 수 있습니다. 선택 항목은 다음과 같습니다.
    • 알파벳순
    • 활성화 된 날짜 (최근 먼저)
    • 활성화 날짜 (가장 오래된 것부터)
    • 총 플레이
    • 후행 주 재생
    • 시작일 (오래된 것부터)
    • 시작일 (최근 먼저)
  • 제한 -재생 목록에 포함 할 최대 동영상 수입니다.
  • 시사 -현재 선택한 옵션으로 작동하는 재생 목록을 보려면 여기를 클릭하십시오.
  • 이 재생 목록의 비디오 -재생 목록의 현재 동영상이 여기에 표시됩니다.
  • 창조하다 -변경 사항을 적용하고 새 재생 목록을 만드는 과정을 시작합니다.

기존 재생 목록 편집

기존 재생 목록을 편집하려면 재생 목록 관리 페이지의 맨 오른쪽 열에있는 비디오 이름 또는 편집 링크를 클릭합니다. 새 재생 목록 추가 페이지와 거의 동일한 페이지로 이동하지만 "만들기"버튼 대신 "저장"버튼이 있습니다. 재생 목록 유형을 제외한 거의 모든 항목을 편집 할 수 있습니다. 유형을 변경하고 저장할 수 있지만 재생 목록을 다시 보면 변경 사항이 실제로 저장되지 않습니다.

경험담

경험을 보려면 경험 관리 탭을 클릭하십시오. 사용 가능한 경험이 나열되고 사용할 수 있습니다. 이름을 클릭하여 볼 수 있으며 경험을 페이지에 포함 할 수 있지만 Brightcove Studio에서만 편집 할 수 있습니다. 경험이 데스크톱과 모바일 모두에서 어떻게 보이는지, 다양한 재생 상태에서 어떻게 보이는지 볼 수있는 옵션이 있습니다.

SharePoint 사이트에 비디오 추가

최신 UI를 사용하는지 아니면 클래식 UI를 사용하는지에 따라 SharePoint 사이트에 비디오를 포함하는 여러 가지 방법이 있습니다. 임베딩 프로세스는 다르지만 두 UI에서 비디오, 재생 목록 및 경험을 임베드하는 프로세스는 동일합니다. Brightcove 비디오 아이콘 대신 Brightcove 재생 목록 또는 Brightcove Experience 아이콘을 클릭하기 만하면됩니다.

최신 UI

최신 UI에서 먼저 더하기 기호를 클릭하여 원하는 위치에 새 앱 파트를 추가합니다. Brightcove 비디오를 선택합니다.

선택 창이 열립니다. 비디오를 선택한 다음 아래로 스크롤하여 적용을 클릭하십시오.

앱 부분

App Part를 추가하려면 App Part 아이콘을 클릭하십시오.

다른 앱 파트에 대한 옵션을 제공하는 메뉴가 열립니다. Brightcove 비디오 옵션을 선택하십시오.

이렇게하면 페이지에 웹 파트가 추가되지만 처음에는 첨부 된 비디오가 없습니다.

확인란을 선택하고 화살표를 클릭하여 "웹 파트 편집"을 선택합니다. 그러면 Brightcove 비디오가 열리고 비디오를 선택할 수 있습니다. 하단의 적용을 클릭합니다. 페이지를 저장하면 비디오가 나타납니다.

웹 파트

웹 파트는 두 가지 방법 중 하나로 추가 할 수 있습니다. 웹 파트 아이콘을 클릭하면 앱 파트 메뉴와 유사한 메뉴가 표시되며 동일한 단계를 수행하면 동일한 결과를 얻을 수 있습니다.

Brightcove 비디오 단추를 다시 클릭하고 비디오를 선택할 수도 있지만 이번에는 "Dowload Web Part"를 선택합니다.

파일이 컴퓨터에 다운로드됩니다. 웹 파트 아이콘을 클릭합니다. 메뉴가 열리면 "웹 파트 업로드"필드에 파일을 업로드합니다. 업로드를 클릭하십시오.

페이지에 동영상이 표시됩니다.
