페이지 내용

    셰어포인트 브라이트코브 커넥터 4.0/4.1: 커넥터 사용

    비디오 추가 및 편집

    콘텐츠 작성자는 SharePoint 커넥터를 사용하여 로컬 컴퓨터에서 비디오를 업로드하고 비디오의 메타데이터를 편집할 수 있습니다.

    동영상 추가

    커넥터의 비디오 추가 섹션은 Video Cloud에 새 비디오를 업로드하는 데 사용됩니다. 사용자가 비디오 추가에 액세스하면 (권한이 있는 경우) Video Cloud로 전송될 메타데이터를 입력할 수 있는 다음 필드가 표시됩니다.

    • 계정 선택 - 이 드롭다운 목록은 사용자가 동영상을 추가할 수 있는 권한이 있는 모든 계정으로 채워집니다. 계정 그룹 할당을 통해 활성화됩니다. 사용자는 업로드를 활성화하려면 계정을 선택해야 합니다.
    • 비디오 파일 - 비디오 파일을 상자로 드래그하거나 “찾아보기”버튼을 클릭하여 로컬 파일 시스템 탐색기를 열어 업로드 할 비디오 파일을 선택하십시오. 화면을 전환하지 않고도 여러 번 반복 할 수 있습니다. 이 화면에서 설정하려는 경우 폴더 및 브라이트코브 태그 다음에 이 단계를 수행해야 합니다. 나중에 편집할 수도 있습니다.
    • 폴더 - 비디오를 조직 목적으로 특정 폴더에 저장할 수 있습니다.
    • 브라이트코브 태그 - SharePoint에 저장될 뿐만 아니라 비디오 클라우드 태그 정보에 저장될 데이터 태그입니다. 이들은 일부 클라이언트가이 인터페이스에 추가하려는 SharePoint 메타 데이터 태그와 별개이며 SharePoint에만 저장됩니다 (비디오 클라우드로 전송되지 않음).

    비디오 처리가 완료되면 사용자가 메타데이터를 추가할 수 있는 “비디오 편집” 링크가 나타납니다.

    동영상 편집

    비디오 편집 화면은 업로드 후 비디오 편집 링크를 통해 액세스하거나 비디오 관리로 이동한 다음 원하는 비디오를 클릭하여 액세스할 수 있습니다.

    사용자는 “사용 가능한 비디오”필드에 검색 단어를 입력하고 데이터 정렬 방법을 선택하여 비디오를 검색 할 수도 있습니다. 비디오의 이름이나 “비디오 편집”링크를 클릭하면 사용자가 비디오 편집 페이지로 이동합니다.

    동영상이 업로드되는 브라이트코브 Video Cloud 계정은 변경할 수 없지만 동영상 이름 및 설명, 계정에 포함된 사용자 정의 필드 등 대부분의 다른 정보는 변경할 수 있습니다. 이름은 파일 이름에서 자동으로 생성되지만 변경할 수 있습니다.

    포스터와 썸네일도 생성되지만 여기에서도 변경할 수 있습니다. 이미지를 변경하는 방법에는 두 가지가 있습니다. 동영상의 오른쪽 상단 모서리에있는 캡처 버튼을 사용하여 비디오에서 새로운 스크린 캡처를 캡처하거나 단순히 새 이미지를 업로드 할 수 있습니다.

    텍스트 트랙 추가

    텍스트 트랙은 첫 번째 업로드 또는 후속 업데이트 시 모든 비디오에 추가할 수 있습니다. 비디오에는 하나 이상의 텍스트 트랙이 연결되어 있을 수 있습니다. 사용자는 비디오에 추가 할 텍스트 트랙의 유형을 선택할 수 있습니다 (파일 업로드, URL 업로드 또는 원격 URL). “없음”을 선택하면 섹션에 필드가 표시되지 않습니다.

    중요 사항: 단일 동영상은 파일 업로드, URL 업로드 또는 원격 URL의 혼합을 지원하지 않습니다. 각 비디오에는 한 가지 유형의 텍스트 트랙만 연결할 수 있습니다.

    파일 업로드

    텍스트 트랙 파일에 대해 “파일 업로드”를 선택하면 다음 필드가 표시됩니다.

    1. 텍스트 트랙 유형 선택 - 파일 업로드, URL 업로드 또는 원격 URL을 선택할 수 있습니다.
    2. 파일 선택기 - 텍스트 트랙 파일의 선택.
    3. 텍스트 트랙 속성 - 텍스트 트랙과 관련된 속성을 설정합니다. “추가”를 클릭하면 다음 업로드/업데이트 시 비디오 데이터에 텍스트 트랙이 추가됩니다.
      1. 언어 - 텍스트 트랙의 언어에 대한 두 글자 코드 (유효한 BCP 47 언어 태그) (예: 영어의 경우 “en”) 입니다.
      2. 레이블 (Label) - 자막에 사용할 수 있는 다양한 언어를 나열하는 메뉴와 같이 사용자에게 표시되는 트랙의 레이블입니다.
      3. 종류 - 나열된 다섯 개 지원 트랙 유형 중 하나
    4. 추가/대기열에 추가된 텍스트 트랙목록 - 다음 저장 시 업로드하기 위해 대기열에 있는 기존 텍스트 트랙 및 새 텍스트 트랙의 목록을 표시합니다. 텍스트 트랙은 “제거” 링크를 클릭하고 비디오를 저장하여 삭제할 수 있습니다.

    URL 업로드, 원격 URL 업로드

    이러한 옵션 중 하나를 선택하면 “파일 업로드” 업로드가 텍스트 트랙 파일에 정규화된 URL 경로를 넣을 텍스트 필드로 바뀝니다.

    • URL 업로드 - 텍스트 트랙 파일이 있는 공용 URL을 입력한 다음 업로드를 누릅니다. 파일이 비디오 클라우드에 업로드됩니다.
    • 원격 URL - 텍스트 트랙 파일이 있는 공용 URL을 입력합니다. URL은 250자 미만이어야 합니다. 캡션은 원격 URL에서 로드됩니다.
    1. 텍스트 트랙 유형 선택 - 파일 업로드, URL 업로드 또는 원격 URL을 선택할 수 있습니다.
    2. URL 추가 - 텍스트 트랙 파일에 대한 정규화된 URL 경로를 입력하는 필드입니다.
    3. 텍스트 트랙 속성 - 텍스트 트랙과 연결된 속성을 설정할 수 있습니다. “추가”를 클릭하면 다음 업로드/업데이트 시 비디오 데이터에 텍스트 트랙이 추가됩니다.
      1. 언어 - 텍스트 트랙의 언어에 대한 두 글자 코드 (유효한 BCP 47 언어 태그) (예: 영어의 경우 “en”) 입니다.
      2. 레이블 (Label) - 자막에 사용할 수 있는 다른 언어를 나열하는 메뉴와 같이 사용자에게 표시될 트랙의 레이블입니다.
      3. 종류 - 나열된 다섯 개 지원 트랙 유형 중 하나.
    4. 추가/대기열에 추가된 텍스트 트랙목록 - 다음 저장 시 업로드하기 위해 대기열에 있는 기존 텍스트 트랙 및 새 텍스트 트랙의 목록을 표시합니다. 텍스트 트랙은 “제거” 링크를 클릭하고 비디오를 저장하여 삭제할 수 있습니다.

    재생 목록 및 환경 추가, 가져오기 및 삭제

    Connector의 재생 목록 관리 섹션은 새 재생 목록을 만들고 편집 및 업데이트를 포함하여 기존 재생 목록을 탐색하고 관리하는 데 사용됩니다. 비디오와 달리 재생 목록 정보는 SharePoint에 저장되지 않고 이 섹션에서 계정을 탐색할 때마다 Video Cloud에서 검색됩니다. 이러한 이유로 API를 통한 재생 목록 데이터의 가용성에 내재된 지연이 있습니다. 이 지연은 다음 섹션에서 호출됩니다.

    재생 목록 관리 소개 페이지

    재생 목록 관리 랜딩 페이지에는 재생 목록을 찾아보거나 사용 가능한 계정에서 새 재생 목록을 만드는 데 사용할 수 있는 필드 목록이 표시됩니다.

    • 계정 선택 - 이 드롭다운 목록은 사용자에게 액세스 권한이 부여된 모든 계정으로 채워집니다. 계정 그룹 할당을 통해 활성화됩니다. 재생 목록 목록을 보려면 사용자가 계정을 선택해야 합니다.
    • 사용 가능한 재생 목록 - 재생 목록을 검색하려면 이 필드에 용어를 입력합니다.
    • 재생 목록 표시 열 - 재생 목록 데이터가 이러한 열에 표시됩니다. 열 머리글을 클릭하여 임의의 열을 기준으로 결과를 정렬할 수 있습니다.
    • 재생 목록 목록 - 각 재생 목록 목록에는 다음이 포함됩니다.
      • 재생 목록 이름 - 이름을 클릭하면 데이터 편집을 위해 재생 목록이 열립니다.
      • 재생 목록에서 활성 상태인 비디오 ID의 목록입니다.
      • 간단한 설명.
      • 재생 목록 유형 - 수동 또는 스마트 재생 목록.
      • 재생 목록의 ID (비디오 클라우드 재생 목록 ID).
      • “재생 목록 편집” 링크입니다. 이 링크 (또는 이름 클릭) 는 편집을 위해 재생 목록을 열거나 세부 정보를 보는 데 사용됩니다.
    • 새 재생 목록 추가 - 선택한 계정에 새 재생 목록을 만들려면 이 단추를 클릭합니다.

    새 수동 재생 목록 추가

    계정에 새 재생 목록을 추가하려면 사용자는 “새 재생 목록 추가”버튼을 클릭합니다. 그러면 재생 목록 추가 양식이 열립니다.

    이동할 비디오를 선택하고 “비디오 이동” 옵션을 활성화하여 재생목록에서 동영상을 위아래로 이동할 수 있습니다. 한 번에 여러 동영상을 이동할 수 있습니다.

    • 이름 - 새 재생 목록의 이름입니다.
    • 설명 - 재생 목록에 대한 간략한 설명입니다.
    • 유형 - 재생 목록의 유형입니다. 이 경우 수동으로 설정해야 합니다.
    • 사용 가능한 비디오 - 검색 상자 및 계정에서 사용할 수있는 비디오 목록을 표시합니다. 이렇게 하면 사용자가 재생 목록에 추가할 비디오 선택 범위를 좁힐 수 있습니다.
    • 작업 화살표 - 선택한 동영상을 재생 목록 안팎으로 이동합니다.
    • 이 재생 목록의 동영상 - 재생 목록의 현재 동영상이 여기에 표시됩니다.
    • 만들기 - 변경 사항을 커밋하고 새 재생 목록을 만드는 프로세스를 시작합니다.

    새 스마트 재생 목록 추가

    • 이름 - 새 재생 목록의 이름입니다.
    • 설명 - 재생 목록에 대한 간략한 설명입니다.
    • 유형 - 재생 목록의 유형입니다. 이 경우 Smart로 설정해야합니다.
    • 태그 - 스마트 재생 목록의 경우 사용자는 재생 목록을 필터링하는 데 사용할 수 있는 하나 이상의 태그를 선택할 수 있습니다. 첫 번째 드롭 다운은 사용자가 “하나 이상 포함”또는 “모두 포함”을 선택할 수 있으며 두 번째 드롭 다운은 사용자가 스마트 재생 목록의 태그를 입력 할 수 있습니다. 선택에 따라 태그가 있는 모든 비디오가 나타나거나 모든 태그가 포함된 비디오만 표시됩니다.
    • 순서 -이 사용자가 일치하는 비디오의 순서를 설정하는 방법을 지정할 수 있습니다. 선택 항목은 다음과 같습니다.
      • 알파벳순
      • 활성화 날짜 (최신 날짜)
      • 활성화 날짜 (가장 오래된 날짜)
      • 총 재생 횟수
      • 주간 재생 횟수
      • 시작 날짜 (가장 오래된 첫 번째)
      • 시작 날짜 (최신 날짜)
    • 제한 - 재생 목록에 포함할 동영상의 최대 양입니다.
    • 미리 보기 - 선택한 옵션과 함께 현재 작동하는 재생 목록을 보려면 여기를 클릭하십시오.
    • 이 재생 목록의 동영상 - 재생 목록의 현재 동영상이 여기에 표시됩니다.
    • 만들기 - 변경 사항을 커밋하고 새 재생 목록을 만드는 프로세스를 시작합니다.

    기존 재생 목록 편집

    기존 재생 목록을 편집하려면 재생 목록 관리 페이지의 맨 오른쪽 열에 있는 동영상 이름 또는 편집 링크를 클릭합니다. 이렇게 하면 새 재생 목록 추가 페이지와 거의 동일한 페이지로 이동하지만 “만들기” 버튼 대신 “저장” 버튼이 있습니다. 재생 목록 유형을 제외하고 거의 모든 것을 편집 할 수 있습니다. 유형을 변경하고 저장할 수 있지만 재생 목록을 다시 보면 변경 내용이 실제로 저장되지 않습니다.

    환경

    경험을 보려면 경험 관리 탭을 클릭합니다. 사용 가능한 경험이 나열되어 사용할 수 있습니다. 이름을 클릭하여 볼 수 있으며 Experience 는 페이지에 포함될 수 있지만 브라이트코브 스튜디오에서만 편집할 수 있습니다. 데스크탑과 모바일에서 경험의 모습과 다양한 재생 상태에서 어떻게 보이는지 볼 수 있는 옵션이 있습니다.

    SharePoint 사이트에 비디오 추가

    최신 UI를 사용하는지 클래식 UI를 사용하는지 여부에 따라 SharePoint 사이트에 비디오를 포함시키는 여러 가지 방법이 있습니다. 임베딩 프로세스는 다르지만 두 UI에서 비디오, 재생 목록 및 경험을 포함하는 프로세스는 동일합니다. 브라이트코브 비디오 아이콘 대신 브라이트코브 재생 목록이나 브라이트코브 경험 아이콘을 클릭하기만 하면 됩니다.

    최신 UI

    모던 UI에서 먼저 더하기 기호를 클릭하여 원하는 위치에 새 앱 파트를 추가합니다. 브라이트코브 비디오를 선택합니다.

    그러면 선택 창이 열립니다. 비디오를 선택한 다음 아래로 스크롤하여 적용을 클릭합니다.

    비디오가 즉시 나타나지 않지만 “게시”또는 “초안으로 저장”을 클릭하면 비디오가 포함 된 완성 된 페이지가 표시됩니다.

    클래식 UI

    모던 UI에는 한 가지 방법으로 임베드할 수 있지만 클래식 UI에는 세 가지 옵션이 있습니다. 이 모든 것은 상단 도구 모음의 맨 오른쪽에있는 “삽입”메뉴에서 제공됩니다.

    포함된

    먼저 브라이트코브 비디오 아이콘을 클릭합니다. 또는 플레이리스트 또는 경험을 삽입하는 경우 해당 항목을 클릭합니다. 프로세스는 동일합니다.

    그러면 브라이트코브 비디오 선택 메뉴가 열립니다.

    비디오를 선택한 다음 아래로 스크롤하여 '클립보드로 복사'를 클릭합니다.“

    브라이트코브 비디오 윈도우를 닫은 다음 삽입 메뉴에서 “임베드”를 클릭합니다.

    복사한 코드를 상자에 붙여넣은 다음 삽입을 클릭합니다.

    동영상이 페이지에 나타납니다.

    앱 파트

    앱 파트를 추가하려면 앱 파트 아이콘을 클릭합니다.

    그러면 다른 앱 파트에 대한 옵션을 제공하는 메뉴가 열립니다. 브라이트코브 비디오 옵션을 선택합니다.

    그러면 페이지에 웹 파트가 추가되지만 처음에는 비디오가 첨부되지 않습니다.

    확인란을 선택하고 화살표를 클릭하여 “웹 파트 편집.“ 그러면 브라이트코브 비디오가 열리고 비디오를 선택할 수 있습니다. 하단에서 적용을 클릭합니다. 페이지를 저장하면 비디오가 나타납니다.

    웹 파트

    웹 파트는 두 가지 방법 중 하나를 추가 할 수 있습니다. 웹 파트 아이콘을 클릭하면 앱 파트 메뉴와 유사한 메뉴가 나타나며 동일한 단계를 수행하면 동일한 결과를 얻을 수 있습니다.

    브라이트코브 비디오 버튼을 다시 클릭하고 비디오를 선택할 수도 있지만 이번에는 “웹 파트 다우로드”를 선택합니다.“

    파일이 컴퓨터에 다운로드됩니다. 웹 파트 아이콘을 클릭합니다. 메뉴가 열리면 파일을 “웹 파트 업로드”필드에 업로드하십시오. 업로드를 누릅니다 .

    동영상이 페이지에 나타납니다.
