Adobe-AEM-Brightcove 커넥터: 설치

이 항목에서는 AEM (Adobe Experience Manager) 용 Brightcove 커넥터 설치에 대해 설명합니다.

소개

Adobe-AEM-Brightcove 커넥터를 사용하면 AEM 내에서 Brightcove Video Cloud 비디오 및 플레이어를 관리하고 AEM 페이지에 비디오를 쉽게 포함 할 수 있습니다.

이 가이드에서는 AEM 6.3을 사용합니다. 커넥터는 AEM 6.2, 6.4 및 6.5에서도 작동하며 단계는 비슷합니다.

시작하기

AEM 개발 인스턴스가 아직 없는 경우 AEM 빠른 시작.jar 파일을 사용하여 인스턴스를 쉽게 만들 수 있습니다.

java  -Xmx1400M -jar AEM_6.3_Quickstart.jar

이 빠른 시작을 사용하는 경우 AEM 인스턴스는 기본적으로 http://localhost:4502 에서 호스팅됩니다. 다음 섹션에서는이 기본 위치를 가정하므로 다른 위치 또는 포트를 사용하는 경우 단계에 표시된 URL을 조정하거나 대신 AEM 탐색 메뉴를 사용해야합니다.

커넥터 설치

다음 단계는 클래식 인터페이스를 사용하여 커넥터를 설치하는 과정을 안내합니다.

  • https://github.com/brightcove/Adobe-AEM-Brightcove-Connector/releases/로 이동하여 사용 중인 AEM 버전의 커넥터 (브라이트코브-올... zip 파일) 를 다운로드합니다.
  • AEM의 개발 인스턴스를 시작합니다.
  • 로그인한 후 CRX 패키지 관리자 (http://localhost:4502/crx/packmgr/index.jsp) 로 이동합니다.
  • 업로드 버튼을 클릭합니다.
  • 업로드 대화 상자에서 커넥터의 zip 파일을 찾아서 확인을 클릭하여 업로드하십시오.
  • 패키지가 목록에 나타나면 설치를 클릭하여 설치합니다.
  • 콘솔 번들 페이지 (http://localhost:4502/system/console/bundles) 로 이동하여 패키지가 활성 상태인지 확인합니다.

이 단계를 완료하면 커넥터가 성공적으로 설치되고구성으로 진행해야 합니다 .
