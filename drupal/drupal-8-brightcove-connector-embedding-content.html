Drupal 8-Brightcove 커넥터: 콘텐츠 임베딩

소개

Drupal-Brightcove 커넥터를 사용하면 Drupal 내에서 Brightcove Video Cloud 비디오 및 플레이어를 관리하고 Drupal 페이지에 비디오를 쉽게 포함 할 수 있습니다. 이것은 Drupal 8 용입니다. 드루팔 7의 경우드루팔 7 지침을 참조하십시오.

Drupal 설치는 매우 쉽게 잘못되는 복잡한 프로세스입니다. 계속하기 전에 콘텐츠가 백업되었는지 확인하십시오.

참고: 이 문서에서는 브라이트코브가 이미 Drupal에 대해 설치 및 구성되었다고 가정합니다. 아직 설치하지 않은 경우 다시 돌아가서 브라이트코브 확장 프로그램을설치및 구성하십시오.

이 페이지에서는 Drupal 콘텐츠에 Brightcove 비디오를 추가하는 방법에 중점을 두지 만 동일한 단계에 따라 재생 목록을 추가 할 수 있습니다.

설정

다음으로 이동합니다. 연장하다 섹션을 찾아 엔티티 브라우저 기준 치수. 확인란을 선택하고 설치를 클릭합니다. 모듈을 찾는 가장 쉬운 방법은 검색 창에 이름을 입력하는 것입니다. 또한 설치 Chaos Tools, Entity Browser IEF, Embed, Entity Embed, 과 CK에디터 모듈

Brightcove 선택 위젯

비디오를 탐색하려면 사용자 정의보기를 설정하십시오. 클릭 구조 탭을 클릭한 다음 견해.

딸깍 하는 소리 보기 추가.  이것은 당신을 데려 갈 것입니다. 추가하다 보다 페이지. 보기에 이름을 지정한 다음 브라이트코브 비디오 보기 설정 메뉴에서 딸깍 하는 소리 저장 및 편집.

다음을 클릭합니다. +추가 에서 디스플레이 섹션 및 선택 엔티티 브라우저.  그런 다음 히트 저장.

형식에 대해 다음을 선택합니다. 테이블 클릭하여 기본 옵션 및 선택 테이블.

이제 브라우저에 대한 필드를 추가 할 차례입니다. 그것이 말하는지 확인하십시오 엔티티 브라우저 구석에. 선택한 후 자동으로 전환되어야합니다.

필드를 추가하려면 추가하다 옆에 있는 버튼 필드 헤더. 필드는 대체로 취향의 문제일 수 있지만 브라이트코브 비디오: 엔터티 브라우저 대량 선택 양식 비디오를 선택할 수 있도록 옵션으로. 이름과 썸네일도 가지고 있으면 좋습니다. 아주 좋은 한 분야 아니 have는 기본적으로 입력한 것입니다. 렌더링된 엔티티.  렌더링된 엔터티는 전체 비디오 및 모든 메타 정보이며 엔터티 브라우저에 포함됩니다. 이로 인해 부피가 크고 비디오를 빠르게 탐색하는 데 유용하지 않습니다.

치는 것을 잊지 마세요 저장.

엔티티 브라우저 생성

엔터티 브라우저를 설정하려면 구성 를 선택하고 엔티티 브라우저 옵션.

결과 페이지에서 엔티티 브라우저 추가 상단에. 결과 페이지에서 새 위젯에 대한 레이블을 추가 한 다음 선택하십시오. 선택하다 아이프레임 디스플레이 플러그인의 경우 탭 위젯 선택기 플러그인의 경우 선택 표시 없음 선택 표시 플러그인의 경우.

다음을 누르십시오. 다음 몇 페이지에는 훌륭한 기본 옵션이 있거나 옵션이 없습니다. 라는 단어가 있는 최종 화면에 도달할 때까지 계속하십시오. 마치다 다음 대신 하단에. 선택하다 보다 에서 위젯 플러그인 메뉴 추가, 그런 다음 위젯에 레이블을 지정합니다. 에 디스플레이 보기, 선택하다 브라이트코브 비디오 브라우저: 엔티티 브라우저.

업로드 탭 추가

클릭 위젯 플러그인 추가 드롭다운 메뉴를 다시 선택하고 엔터티 양식.

그러면 첫 번째 양식 아래에 양식이 추가됩니다. 레이블을 추가하고 선택 브라이트코브 비디오 위해 엔티티 유형 과 묶음. 히트 마치다.

Brightcove 버튼 추가

로 이동 구성. 에서 콘텐츠 저작 섹션, 선택 텍스트 편집기 삽입 버튼.  

딸깍 하는 소리 삽입 추가 단추. 버튼에 레이블을 지정하고 선택하십시오. 실재 포함 유형 및 브라이트코브 비디오 엔터티 유형의 경우. 엔티티 브라우저는 브라이트코브 콘텐츠에 대해 이미 생성한 브라우저여야 합니다.

아이콘이 필요하면 여기에 업로드해야합니다. 그렇지 않으면 기본 버튼 아이콘이됩니다. 다음은 사용할 수있는 작은 Brightcove 로고 아이콘입니다.

로 돌아가기 구성 그리고 선택 텍스트 형식 및 편집기.

엔터티 포함 단추를 추가 할 텍스트 형식을 구성하도록 선택합니다. 게시 할 때이 텍스트 형식을 사용해야합니다. 이 과정을 반복하여 버튼을 여러 형식에 추가 할 수 있습니다.

확실하게하다 씨에디터 에 대해 선택됩니다. 텍스트 에디터. 새 Brightcove 버튼이 사용 가능한 버튼로 드래그할 수 있습니다. 활성 도구 모음.

확실하게하다 포함된 엔터티 표시 선택하고 클릭 구성 저장.

비디오가 제대로 표시되도록하려면 먼저 새 Brighcove보기를 만들어야합니다. 그렇게 하려면 다음으로 이동하십시오. 구조 페이지를 선택하고 디스플레이 모드. 그런 다음 보기 모드.

딸깍 하는 소리 새 보기 추가. 선택하다 브라이트코브 비디오 보기 모드 엔터티 유형으로. 새 보기에 레이블을 지정하고 저장.

이제 Brightcove Video 아래에 나열된보기가 표시됩니다.

다음으로 새보기 모드를 활성화합니다. 이동 브라이트코브 비디오 설정 에서 구조 메뉴를 선택한 다음 디스플레이 탭 관리. 하단에는 사용자 지정 디스플레이 설정 상자. 새 사용자 정의 보기를 선택하고 저장.

기본적으로보기에는 비디오와 함께 표시되는 불필요한 메타 데이터가 많이 있습니다. 옆에 기본 보기 링크에서 보기 모드 이름이 있는 링크를 클릭합니다. 게시물에 게시하지 않을 메타 데이터를 비활성화하고 유지하려는 메타 데이터를 활성화합니다. 저장을 누르는 것을 잊지 마십시오.

마지막으로 다음으로 이동하여 캐시를 지우십시오. 공연 에서 구성 메뉴를 클릭하고 모든 캐시 지우기.
