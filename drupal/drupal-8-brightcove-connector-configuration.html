    드루팔 8 브라이트코브 커넥터: 배열 형태

    서문

    드루팔 브라이트코브 비디오 커넥터를 사용하면 드루팔 내에서 브라이트코브 비디오 클라우드 비디오 및 플레이어를 관리하고 드루팔 페이지에 비디오를 쉽게 삽입할 수 있습니다. 이것은 드루팔 8을위한 것입니다. 드루팔 7의 경우드루팔 7 지침을 참조하십시오.

    드루팔 (Drupal) 설치는 매우 쉽게 잘못된 복잡한 과정입니다. 계속하기 전에 콘텐츠가 백업되었는지 확인합니다.

    브라이트코브 계정에 연결

    1. Brightcove 계정을 연결하려면 다음을 클릭하여 Brightcove API 페이지로 이동하십시오. 구성을 누른 다음 아래로 스크롤하여 Brightcove API 계정 아래 옵션미디어 . 그러면 브라이트코브 API 클라이언트 페이지로 이동합니다. 새 Brightcove API 클라이언트를 추가하려면 Brightcove API 클라이언트 추가 .
    2. 레이블은 무엇이든 가능하지만 브라이트코브의 계정 이름과 동일한 이름을 사용하면 쉽게 식별할 수 있습니다.
    3. 클라이언트 ID 및 클라이언트 암호를 얻으려면 응용 프로그램을 등록해야합니다. 브라이트코브 스튜디오에서API 인증으로이동한 다음새 애플리케이션 추가를클릭합니다.
    4. 이름과 간단한 설명을 입력합니다.
    5. 계정을 선택합니다. 아래 그림은 모든 필수 권한입니다.
    6. 저장을 누르십시오.
    7. 클라이언트 ID 및 클라이언트 암호가 나타납니다. 클라이언트 암호를 복사하여 안전한 위치에 저장해야 합니다. 다시 액세스할 수 없습니다.
    8. 계정 ID를 찾으려면 브라이트코브 스튜디오 대시보드의 왼쪽 상단에서 확인하세요.
    9. 모든 ID와 키를 입력하고기본 플레이어 .
    10. 저장을 클릭합니다. 이제 API 클라이언트가 목록에 나타납니다.

    동기화

    처음에는새로운 콘텐츠페이지 및 클릭 Brightcove 동영상 , 동영상을 추가 할 수있는 옵션이 있지만 아무 것도 없습니다. 먼저 데이터를 동기화해야하기 때문입니다. 초기 동기화 후, 드루팔의 chron은 모든 것을 최신 상태로 유지합니다. 수동 동기화의 경우보고서버튼을 클릭 한 다음 Brightcove 상태 개요 .

    에 Brightcove 상태 개요페이지, 히트모두 동기화 .

    브라이트코브 동영상 페이지로 다시 이동하면 동영상을 사용할 수 있습니다.

    사용 권한 구성

    이제 비디오를 보고 게시할 수 있지만 관리자가 아닌 사용자가 비디오와 상호 작용할 경우 권한을 설정해야 합니다. 이러한 권한을 설정하려면 먼저 다음을 클릭하여 권한 페이지로 이동합니다. 사람들 그리고권한 .

    아래로 스크롤하여 브라이트코브 권한을 찾습니다. 필요에 따라 활성화 및 비활성화하십시오. 브라이트코브 권한은 자동으로 관리자 전용으로 설정되므로 다른 사용자가 동영상을 볼 수 있도록 하려면 해당 권한을 부여해야 합니다.
