Drupal 8-Brightcove 커넥터: 구성

소개

Drupal Brightcove 비디오 커넥터를 사용하면 Drupal 내에서 Brightcove Video Cloud 비디오 및 플레이어를 관리하고 Drupal 페이지에 비디오를 쉽게 포함 할 수 있습니다. 이것은 Drupal 8 용입니다. 드루팔 7의 경우드루팔 7 지침을 참조하십시오.

Drupal 설치는 매우 쉽게 잘못되는 복잡한 프로세스입니다. 계속하기 전에 콘텐츠가 백업되었는지 확인하십시오.

Brightcove 계정에 연결

  1. Brightcove 계정을 연결하려면 다음을 클릭하여 Brightcove API 페이지로 이동합니다. 구성, 그런 다음 아래로 스크롤하여 브라이트코브 API 계정 아래의 옵션 미디어. Brightcove API 클라이언트 페이지로 이동합니다. 새 Brightcove API 클라이언트를 추가하려면 Brightcove API 클라이언트 추가.
  2. 레이블은 무엇이든 될 수 있지만 Brightcove의 계정 이름과 동일한 이름을 사용하면 쉽게 식별 할 수 있습니다.
  3. 클라이언트 ID와 클라이언트 암호를 얻으려면 응용 프로그램을 등록해야합니다. Brightcove Studio에서 다음으로 이동합니다. API 인증 그리고 클릭 새 애플리케이션 추가.
  4. 이름과 간단한 설명을 입력하십시오.
  5. 계정을 선택하십시오. 아래 그림은 필요한 모든 권한입니다.
  6. 저장을 누르십시오.
  7. Client ID와 Client Secret이 나타납니다. 클라이언트 시크릿을 안전한 위치에 복사하여 저장하십시오. 다시 액세스 할 수 없습니다.
  8. 계정 ID를 찾으려면 관리 페이지의 계정 정보 섹션으로 이동하십시오.
  9. 모든 ID와 키를 입력하고 선택 기본 플레이어.
  10. 딸깍 하는 소리 저장. 이제 API 클라이언트가 목록에 나타납니다.

동기화

처음에 다음으로 이동하면 새로운 콘텐츠 페이지 및 클릭 브라이트코브 동영상, 비디오를 추가하는 옵션이 있지만 거기에는 없습니다. 먼저 데이터를 동기화해야하기 때문입니다. 초기 동기화 후 Drupal의 연대기는 모든 것을 최신 상태로 유지합니다. 수동 동기화의 경우 보고서 버튼을 누른 다음 브라이트코브 상태 개요.

에 브라이트코브 상태 개요 페이지, 히트 모두 동기화.

Brightcove 비디오 페이지로 다시 이동하면 비디오를 사용할 수 있습니다.

권한 구성

이제 비디오를보고 게시 할 수 있지만 관리자가 아닌 사용자가 비디오와 상호 작용하려면 권한을 설정해야합니다. 이러한 권한을 설정하려면 먼저 다음을 클릭하여 권한 페이지로 이동합니다. 사람들 그리고 권한.

아래로 스크롤하여 Brightcove 권한을 찾습니다. 필요에 따라 활성화 및 비활성화합니다. Brightcove 권한은 자동으로 관리자 전용으로 설정되므로 다른 사람이 비디오를 볼 수 있도록하려면 해당 권한을 부여해야합니다.

이제콘텐츠 관리또는 콘텐츠포함으로 넘어갈 수 있습니다.
