    드루팔 7-브라이트코브 커넥터: 구성 및 사용

    서문

    드루팔 브라이트코브 비디오 커넥터를 사용하면 드루팔 내에서 브라이트코브 비디오 클라우드 비디오 및 플레이어를 관리하고 드루팔 페이지에 비디오를 쉽게 삽입할 수 있습니다. 이 페이지에서는 Drupal 7에서 브라이트코브 모듈을 구성하는 방법에 대해 자세히 설명합니다. 드루팔 8의 경우, 드루팔-브라이트코브 커넥터: 구성.

    드루팔 (Drupal) 설치는 매우 쉽게 잘못된 복잡한 과정입니다. 계속하기 전에 콘텐츠가 백업되었는지 확인합니다.

    브라이트코브 계정에 연결

    1. 브라이트코브 계정을 연결하려면구성을클릭한 다음미디어아래의 브라이트코브설정 옵션으로 스크롤하여 브라이트코브 API 페이지로 이동합니다. 그러면 브라이트코브 설정 페이지로 이동합니다. 새 브라이트코브 API 클라이언트를 추가하려면API클라이언트 탭을 클릭한 다음API 클라이언트 추가를선택합니다. 그러면 API 클라이언트에 추가 할 양식으로 이동합니다.
    2. 레이블은 무엇이든 가능하지만 브라이트코브의 계정 이름과 동일한 이름을 사용하면 쉽게 식별할 수 있습니다.
    3. 계정 ID를 찾으려면 브라이트코브 스튜디오 대시보드의 왼쪽 상단에서 확인하세요.
    4. 클라이언트 ID 및 클라이언트 암호를 얻으려면 응용 프로그램을 등록해야합니다. 브라이트코브 스튜디오에서API 인증으로이동한 다음새 애플리케이션 추가를클릭합니다.
    5. 이름과 간단한 설명을 입력합니다.
    6. 계정을 선택합니다. 아래 그림은 모든 필수 권한입니다.
    7. 저장을 누르십시오.
    8. 클라이언트 ID 및 클라이언트 암호가 나타납니다. 클라이언트 암호를 복사하여 안전한 위치에 저장해야 합니다. 다시 액세스할 수 없습니다.
    9. 모든 ID와 키를 입력합니다. 저장을 누르십시오.
    10. 문제가 발생하면 날짜와 시간이 구성되어 있는지 확인한 후 다시 시도하십시오.

    브라이트코브 비디오 필드 추가

    브라이트코브 비디오를 드루팔 게시물에 추가하려면 먼저 새 콘텐츠 페이지에 비디오 필드를 추가합니다. 이렇게하려면 “구조”로 이동하여 “콘텐츠 유형”을 클릭하십시오.“

    브라이트코브 비디오를 추가하려는 콘텐츠 유형에서 “필드 관리”를 선택합니다. 이 프로세스는 각 콘텐츠 유형에 대해 반복될 수 있습니다.

    “새 필드 추가”를 클릭합니다.“ 레이블을 지정하고 필드 유형으로 “브라이트코브”를 선택하고 위젯의 경우 “동영상 (업로드/찾아보기)" 를 선택합니다. 저장을 누르십시오.

    이제 비디오 필드가 나타납니다.

    동영상 게시 및 관리

    드루팔 8과 달리 Drupal 7 커넥터에는 별도의 콘텐츠 관리 시스템이 없으므로 대부분의 콘텐츠 관리는 브라이트코브 스튜디오 사이트에서 수행해야 합니다. 그러나 새 콘텐츠 페이지에서 찾아보기를 눌러 모든 이미지를 볼 수 있으며 새 이미지를 업로드할 수 있습니다.

    기존 동영상을 게시물에 첨부하는 방법에는 두 가지가 있습니다. 동영상 제목을 알고 있다면 브라이트코브 필드에 동영상을 입력하고 자동 완성 기능을 통해 동영상을 찾을 수 있습니다. 찾아보기 버튼을 눌러 모든 동영상을 보고 선택할 수도 있습니다.

    업로드 버튼을 클릭하면 업로드 양식이 열리므로 드루팔에서 바로 새 동영상을 추가 할 수 있습니다. 게시물을 게시하면 동영상이 나타납니다.
