Drupal 7-Brightcove 커넥터: 구성 및 사용

소개

Drupal Brightcove 비디오 커넥터를 사용하면 Drupal 내에서 Brightcove Video Cloud 비디오 및 플레이어를 관리하고 Drupal 페이지에 비디오를 쉽게 포함 할 수 있습니다. 이 페이지는 Drupal 7 내에서 Brightcove 모듈을 구성하는 방법을 자세히 설명합니다. 드루팔 8의 경우드루팔 브라이트코브 커넥터를 참조하십시오. 구성.

Drupal 설치는 매우 쉽게 잘못되는 복잡한 프로세스입니다. 계속하기 전에 콘텐츠가 백업되었는지 확인하십시오.

Brightcove 계정에 연결

  1. Brightcove 계정을 연결하려면 다음을 클릭하여 Brightcove API 페이지로 이동합니다. 구성, 그런 다음 아래로 스크롤하여 브라이트코브 설정 아래의 옵션 미디어. Brightcove 설정 페이지로 이동합니다. 새 Brightcove API 클라이언트를 추가하려면 API 클라이언트 탭을 선택한 다음 API 클라이언트 추가. API 클라이언트에 추가 할 수있는 양식으로 이동합니다.
  2. 레이블은 무엇이든 될 수 있지만 Brightcove의 계정 이름과 동일한 이름을 사용하면 쉽게 식별 할 수 있습니다.
  3. 계정 ID를 찾으려면 브라이트코브 스튜디오 대시보드의 왼쪽 상단에서 확인하십시오.
  4. 클라이언트 ID와 클라이언트 암호를 얻으려면 응용 프로그램을 등록해야합니다. Brightcove Studio에서 다음으로 이동합니다. API 인증 그리고 클릭 새 애플리케이션 추가.
  5. 이름과 간단한 설명을 입력하십시오.
  6. 계정을 선택하십시오. 아래 그림은 필요한 모든 권한입니다.
  7. 저장을 누르십시오.
  8. Client ID와 Client Secret이 나타납니다. 클라이언트 시크릿을 안전한 위치에 복사하여 저장하십시오. 다시 액세스 할 수 없습니다.
  9. 모든 ID와 키를 입력하십시오. 저장을 누르십시오.
  10. 문제가 발생하면 날짜와 시간을 구성했는지 확인하고 다시 시도하십시오.

Brightcove 비디오 필드 추가

Drupal 게시물에 Brightcove 비디오를 추가하려면 먼저 새 콘텐츠 페이지에 비디오 필드를 추가하십시오. 이렇게하려면 "구조"로 이동하여 "콘텐츠 유형"을 클릭하십시오.

Brightcove 비디오를 추가하려는 컨텐츠 유형에서 "필드 관리"를 선택합니다. 이 프로세스는 각 콘텐츠 유형에 대해 반복 할 수 있습니다.

"새 필드 추가"를 클릭하십시오. 레이블을 지정하고 필드 유형에 대해 "Brightcove"를 선택하고 위젯에 대해 "동영상 (업로드 / 찾아보기)"을 선택합니다. 저장을 누르십시오.

이제 비디오 필드가 나타납니다.

비디오 게시 및 관리

Drupal 8과 달리 Drupal 7 커넥터에는 별도의 콘텐츠 관리 시스템이 없으므로 대부분의 콘텐츠 관리는 Brightcove Studio 사이트에서 수행해야합니다. 그러나 새 콘텐츠 페이지에서는 찾아보기를 눌러 모든 이미지를 볼 수 있으며 새 이미지를 업로드 할 수도 있습니다.

게시물에 기존 동영상을 첨부하는 방법에는 두 가지가 있습니다. 비디오 제목을 알고있는 경우 Brightcove 필드에 입력하고 자동 완성 기능이 작동하도록합니다. 찾아보기 버튼을 눌러 모든 비디오를보고 선택할 수도 있습니다.

업로드 버튼을 클릭하면 업로드 양식이 열리므로 Drupal에서 바로 새 동영상을 추가 할 수 있습니다. 게시물을 게시하면 비디오가 나타납니다.
