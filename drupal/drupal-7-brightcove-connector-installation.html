Drupal 7-Brightcove 커넥터: 설치

소개

Drupal-Brightcove 커넥터를 사용하면 Drupal 내에서 Brightcove Video Cloud 비디오 및 플레이어를 관리하고 Drupal 페이지에 비디오를 쉽게 포함 할 수 있습니다. 이 기사에서는 특히 Drupal 7 플러그인을 설치하는 방법에 대해 설명합니다. 드루팔 8의 경우 드루팔 8의 설치 가이드를 참조하십시오.

Drupal 설치는 매우 쉽게 잘못되는 복잡한 프로세스입니다. 계속하기 전에 콘텐츠가 백업되었는지 확인하십시오.

설치 단계

최상의 결과를 얻으려면 이미 composer를 설치했는지 확인하십시오. 방법에 대한 지침은여기에서 확인할 수 있습니다.

  1. PHP-API 래퍼를 다운로드하여 설치합니다. 다음으로 이동sites/all/libraries/PHP-API-Wrapper
  2. sites/all/libraries/PHP-API-Wrapper본체에서 로 이동합니다. 폴더 내에서 다음을 실행하십시오. composer install.
  3. 이제브라이트코브 모듈을 다운로드하십시오. 종속성을 위해 다른 모듈도로드해야합니다.
    • 카오스 도구
    • 엔티티 API
    • 라이브러리 API
    • 데이트
    • 파일 엔티티
    • 미디어
    • 견해
  4. 모든 모듈을modules폴더로 이동합니다.
  5. 드루팔 사이트로 이동하십시오. 모듈 페이지로 이동합니다.
  6. ctools, 엔터티 API, 라이브러리, 날짜, 날짜 API, 날짜 팝업, 미디어, 미디어 인터넷 소스, 파일 엔티티 및 뷰와 같은 모듈을 사용할 수 있습니다. 3 단계 덕분에 모두 사용할 수 있습니다.
  7. 날짜와 시간을 설정하십시오! 그렇지 않으면 Brightcove 모듈이 작동하지 않습니다.

완료되면구성으로 이동할 수 있습니다 .
