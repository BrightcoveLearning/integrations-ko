소개

Microsoft Teams용 Brightcove 비디오 커넥터(비디오 커넥터)는 다음 두 가지 방법 중 하나로 설치할 수 있습니다. 수동으로, Teams 사용자 지정 앱으로 또는 App Store를 통해 (곧 제공 예정).

전제 조건

  • 마이크로소프트 팀즈
  • 비디오 커넥터를 설치할 Teams 위치에 대한 소유자권한

수동(커스텀 앱) 설치

비디오 커넥터를 수동으로 설치하려면(팀에서 사용자 지정 앱 설치) 아래 단계를 따르세요.

  1. 여기 GitHub 리포지토리에서 비디오 커넥터 매니페스트 패키지의 최신 릴리스를 다운로드하십시오. 릴리스 · 오션스베이브/브라이트코브-팀 커넥터 (github.com)를 사용하여 로컬 컴퓨터에 저장합니다.
  2. Microsoft Teams를 열고 커넥터를 설치할 팀을 엽니다.
  3. 추가 옵션메뉴를 선택합니다.
  4. “팀 관리” 메뉴 항목을 클릭합니다 ( 참고 : 언급된 메뉴 항목을 보려면 이 팀 위치에 대한 소유자 권한이 있어야 합니다. 위의 필수 구성 요소 섹션 참조).
  5. 앱탭을 선택합니다.
  6. 앱 페이지 하단에 있는 사용자 지정 앱업로드링크를 클릭합니다.
  7. 커넥터 매니페스트 (이전에 다운로드한) 의 위치로 이동하여 선택하고열기버튼을 클릭합니다.
  8. 딸깍 하는 소리추가하다표시되는 Brightcove Video Connect 모달 창에서:
  9. Video Connector 앱이 설치되어 있어야 하며 이제 아래와 같이 팀에 사용 가능한 앱 목록에 표시되어야 합니다.

이후 섹션에서 커넥터 구성에 대해 다루고 Microsoft Teams용 커넥터사용에서 사용할 것입니다.

마이크로소프트 앱 스토어에서 설치

이 기능은 현재 프로비저닝 중이며 문서가 제자리에 있으면 업데이트될 것입니다.

팀에 Brightcove Media Asset Manager 추가

먼저 미디어 관리자를 추가할 팀에 비디오 커넥터를 설치해야 합니다. 아직 수행하지 않은 경우 설명서의 이전 섹션에 설명된 대로 설치 안내서를 따르십시오.

이제 구성을 진행하겠습니다.

  1. Microsoft Teams에서 이미 비디오 커넥터 앱이 설치된 팀을 엽니다.
  2. 미디어 관리자를 설치하려는 팀 채널을 엽니다.
  3. 탭추가메뉴 옵션을 클릭합니다.
  4. 탭추가모달 창에서브라이트코브 비디오 연결앱을 선택합니다.
  5. 에서구성모달 창, 아래브라이트코브 계정딸깍 하는 소리새로운커넥터에 계정을 추가하거나 이전에 이 Teams 환경의 다른 팀에서 커넥터를 사용한 적이 있는 경우 드롭다운에서 계정을 선택합니다.
  6. 새 브라이트코브 계정을 추가하는 경우 (이전 단계에서새로만들기를 클릭한 경우) 그림과 같이 브라이트코브 계정 ID, API 액세스 키 ID (클라이언트 ID) 및 API 보안 액세스 키 (클라이언트 암호) 로 양식을 작성하십시오. 계정저장 후 계속을 클릭합니다 .

    클라이언트 자격 증명을 가져오는 방법에 대한 자세한 내용은 API 요청에 대한인증을 참조하십시오 . 자격 증명에는 CMS, Dynamic Ingest 및 Ingest Profiles API에 대해 최소한 다음 권한이 필요합니다.

  7. 구성 창의커넥터 탭 모드 아래의 드롭다운옵션에서브라이트코브 에셋 매니저를선택합니다.
  8. 저장을 클릭합니다 .
  9. 이제 Brightcove Asset Manager가 아래와 같이 새로 생성된 탭에 나타납니다.

이제 Brightcove 비디오 커넥터가 설치 및 구성되었습니다. 커넥터를 사용하여 미디어 자산을 관리하려면 Microsoft Teams용 커넥터사용을 참조하십시오 .
